>

## 文件列表

| 檔案 | 說明 |
|------|------|
| [index.html](index.html) | 玩家區入口（含密碼 gate） |

## 不包含的內容

- GM truth / 完整真相
- 主持人流程與時間規劃
- 角色完整秘密（僅各角色自己知道）
- 結局解析
