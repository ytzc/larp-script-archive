公開簡介

《寇俠》是一個古風陣營還原類型的劇本殺 / LARP。

故事以「起朝」為背景，玩家將在「天福客棧」這個江湖交匯之處，扮演各有立場的角色，歷經陣營角力與事件還原。

風格與元素

  • 古風江湖：以起朝為時代背景，融合武俠江湖氣氛
  • 陣營機制：玩家各屬不同陣營，陣營目標影響遊戲走向
  • 還原要素：事件真相的還原是遊戲核心之一
  • 機制設計：包含特定機制設計（詳見遊戲規則說明）

注意事項

本劇本包含陣營對立與武力對抗要素，適合喜歡推理、陣營博弈與角色扮演的玩家。
