你的故事（頁002，接前頁）

任務大意：

你有著一個沉重的心結：你曾奉命在一次行動中傷害過無辜，那件事一直是你心中的刺。如今你已是一個普通農夫，只想平靜度日，守護那些還活著的舊日同袍。

數日前，你收到了一封密信，得知大夏皇族後裔正在籌謀大事，欲在汲洲集結同道之人。這讓你內心翻湧，往日的熱血又開始沸騰，但你已老了，你不想再捲入爭鬥，你只想讓那些舊部好好地活著。

給玩家的提示（頁009上）

你的角色是本場遊戲中武力最高、生命最強的角色之一，但這並不代表你應該一味強攻。農叟的核心是一個老去的武將——他的選擇來自於情義，而非利益。

建議扮演方向：

  • 農叟沉默寡言，做事穩重，話不多但句句有分量。
  • 你對舊部有強烈的保護欲，凡是可能傷害到他們的事，你都會優先阻止。
  • 你的身份是本場的重要秘密，請勿輕易透露「丁崧」這個名字。

在這次汲洲之行中，你本是奉密信所召，想低調打探消息。然而冷降塵的死讓一切亂了陣腳，你不得不捲入其中。你最想知道的事只有一件：那個害死冷降塵的人，究竟是誰，動機是什麼？
