農叟

真名：丁崧　｜　50歲左右

自古名士如美人，不許人間見白頭。

情景一

你的故事

可還曾記得戰場上的金戈鐵馬？你自嘲地搖了搖頭，又用藥鏟對着山巖上那株暗紅色花苞輕輕地挖了下去。這藥材採集頗爲不易，半天下來也只得了僅底這淺淺的一層，且只有苦脊山最險峻處才有那麼零星幾朵而已。蒼脊山，應該離兒子近了那麼一點點吧？ 你生活在盤興口，所有人都喊你農叟……如今你不過是一個隱居的耕農老頭罷了，但是你自己知道，其實你是丁崧，是二十年前那個時代最璀璨的英雄。身爲大夏王朝終末的託孤重臣，你深知自己身上背負的責任有多重。你年少從軍，經歷了大夏王朝末期的衰敗，目睹了權臣鄭木以攝政居天子之位的囂張。眼看着天下局勢越來越混亂，你第一次有了解甲歸田的想法。可是你知道你不能，因爲你還在爲大夏王族做着最後的打算。 在大夏王朝的最後數年，鄭木爲了確保自己登基無恙，便暗自派遣他豢養的特務機構繡衣使前去擊殺全天下的大夏皇族後裔。你與濟陽縣令，同時也是大夏開國皇帝八世孫的何彥世代交好，知道此刻何彥一家老小命在旦夕，便一人一騎千裏奔赴濟陽。一路上絲毫不敢停歇，從京師到濟陽縣有幹裏之遙，你卻只用了三天，可終歸是晚來了一步。 午夜時分，何彥府宅如修羅地獄一般，院落屋中躺着一十八具屍首，其中便包括了何彥夫婦。你心痛萬分，查看了何彥夫婦的屍體，發現兩人是被人數掌擊於心脈左側，心脈斷裂而亡，還未等你仔細觀察其他倒地的家僕的屍體，便看到遠處屋頂上有身影閃過。你立刻追了過去，借着月光發現此人懷中抱着一個女童，正向前狂奔，那女童應當就是何彥的幼女。 你一掌劈過去，那人此時懷中抱着女童，來不及躲閃，右肩處中了你一掌。只聽他悶哼一聲，也顧不得回頭，左手向後投出一陣煙霧。你只覺得一陣異香撲鼻，隨後感到胸悶頭暈。 僅是一眨眼的功夫，那人已經抱着女童遠遠躍開，你再也追不上了，空氣中只留下這迷藥的異香。 你失魂一般地回到京師，你私人部曲的一員叫龐萬金的龐姓小校急急向你匯報，鄭木要於明日召集大夏所有高級將領共同商討剿匪之事。原來在鄭木掃平大夏宗親之後，各地響應倒鄭號召之人如星火燎原，又因爲連年的戰亂、水災、人禍……百姓們實在走投無路，紛紛落草爲寇，而他們也被民間統稱爲：寇俠。鄭木當然將天下寇俠視爲眼中釘，你也被派去討伐寇俠，討伐過程中那姓小將龐萬金表現得異常活躍。 那是一支名爲「鶴須」的寇俠。其實你對他們早有耳聞，鶴須軍雖是草寇，但從來只行仁義之事。若不是上峯有令，你真不想與他們爲敵。數次搏殺後，你抓住了鶴須軍的統領冷佛林，哪曾想此人義薄雲天，對自己被捕之事毫不在意，只是對自己屬下的安危極爲關心。 你對他產生了不小的興趣，一個流寇居然如此大義。你敬他是條漢子，便帶了酒水前去，沒想到此人和你談及如今天下局勢時，頗有見解。雖然你是將，而他是被抓的敵，但你們兩個還是一見如故，互相欣賞。接連數日你都在牢車旁同他促膝長談，分析天下局勢。你更確定了這乃是一個胸懷天下的漢子，最終你做出了一個決定，將冷佛林私放。 冷佛林平安潛走，你卻因此遭受牢獄之災。鄭木得知私放之事後暴怒，以叛國之名將你流放，家人老小等一併充軍，你和全家老小就此失散。鄭木怕你東山再起，便將你一家老小各自流放於不同的地區，並派遣繡衣使中的精英骨幹沿途護送。除此之外，鄭木還解散了你麾下的私人部曲，瓦解了你的勢力。**你動用了一切人脈也只將親子丁典救出，此時你毫無辦法，只能將他託付於冷佛林。臨走時，你雕刻了一把小木劍給兒子作爲玩具⋯⋯ 至今你都想不明白，爲何猶如鐵板一塊的私人部曲，會對外泄露此事，難道是內部有人反水不成？** 次年權臣鄭木改元始建國，改國號爲「起」，寓力新朝起始，朝氣蓬勃之意。起朝大赦天下，你也因此自由。然而在鄭木的統治下，百姓流離失所，民不聊生。你的家人也早已流溶四方，無法重聚。唯一知曉下落的兒子隨着鶴須軍東徵西戰，居無定所。此時你已然知道寇俠之中有不少仁義之士，加之親眼看到天下黎民在起朝統治下的慘狀，你對目前的朝廷更是憤恨有加，對舊朝大夏極爲懷念。但你知道，憑你各人之力是無法改變現狀的 你心灰意冷，來到蒼脊山的盤興口，在村外開墾了一片農田，平日以農耕爲主，到了鶴紅草長成的月份，就上山採藥。你將自己種的糧食和挖掘的藥草在盤興口售賣，換取生活所需之物，徹底成爲了隱居之人。 長久耕作勞苦，你未及五旬卻兩鬢斑白，所以當地人喚你「農叟」。

近日經歷

最近數年，你都會以「農叟」的身份前往盤興口，除了探聽天下局勢之外，還希望能夠獲得更多有關兒子的消息。數年以來，鶴須軍威名大盛，龍瑞二年冷佛林之死的消息傳遍江湖，你也爲之傷懷。你知道龍瑞四年寇俠聯軍雖敗於起朝軍隊，但是鶴須軍在大統領冷佛林之子—冷降塵的帶領下已然恢復生氣。 只是你還不知道自己的兒子丁典如今在鶴須軍中表現如何，你也不敢想兒子是否已在龍瑞四年戰死…… 除了農耕之外，你最重要的事便是在蒼脊山的山崖上挖掘當地赫赫有名的「鶴紅草」。 這種藥物乃是天下藥中之首，據說功效非凡，全天下也僅在蒼脊山的陡壁上生長，普通的採藥人根本無法挖掘，你也是憑藉多年的行伍功夫才能在山間攀援採集。 你本想就此徹底隱居，不曾想此地也潛藏着危機。前日你竟然遇到伏擊，好在你武功高強沒有受傷，但不慎讓殺手逃脫，可借午夜擊退來敵時，你並未看清來人相貌，不知道對方是誰。但此人逃竄時身上落下一物，乃是一枚潤黃的玉牌，看起來價值不菲。 你認出這是寇俠頭目的銘牌，而同自己唯一有關係的便是寇俠鶴須軍，難道暗殺自己的乃是鶴須軍之人？你百思不得其解。自己和鶴須軍前任大統領冷佛林是生死之交，如今鶴須軍竟然派人暗殺你？ 不論怎麼樣，這裡已經不是最佳的隱居之地了。你簡單收抬，只帶了鶴紅草便前往盤興口，這是走出蒼脊山脈的必經之路。 沒想到在盤興口的告示欄中，你看到了這樣一張不同尋常的安民告示，上書：「初夏已至，福自始來，酷熱難避，安談室中。」落款的署名是「人口丁」。對他人來說，這安民告示僅僅是告知民衆在家中避暑的內容，寫得還頗爲古怪。但你知道，這其中的「夏」指的分明是大豆前朝，落款的三個字組合後乃是一個「何」字，正是大夏皇族的姓氏。文中暗指大夏正統不日前來，讓人在屋中等待，那這個所謂的屋，指的必然是其中的「福」字。 你知道，這個安民告示必是大夏皇族血脈留下的，用來召集大夏舊臣。 而整個盤興口帶有「福」字的所在，便只有一個，那就是今年剛剛開業的天福客棧⋯⋯

今日經歷

今日是麒安二年七月十一，你知道，必須前往天福客棧了。 客棧中此時頗爲冷清。老闆娘一襲紅衣，雖年過三十，但風韻猶存，正懶懶靠在案臺上，指使着店小二跑腿兒上茶。大堂內一個婀娜的曲兒姐正在彈唱，她身邊則站着一個肥碩男子，看年紀是父女一起行走江湖賣藝爲生，只是這男子爲何看起來有些面熟？而且看你的眼神還有所躲閃。 他們的對面坐着一對夫妻。男的看着是個白面書生，三十餘歲，看起來病懨懨的，長得倒還不錯，身形莫名有些熟悉，女的頗爲年輕，紅紗罩面，身形卻極是動人。你不禁感嘆，這世道變了，老夫少妻卻如此大膽地出門招搖。 除了他們以外，客棧大堂上還坐着三個客商打扮的男子。爲首的是個大漢，絡腮黑須，雙目炯炯，喝起酒來豪爽無比，你對他頗有好感。大漢身後則是兩個男子，一個氣勢卓絕，一個瘦小乾枯，以黑布蒙面，露出的臉龐上有腐爛的疤痕。 店小二熱情地給你倒了杯茶，但你心中還惦記着那張告示內容，並不敢就此飲下。只隨手將隨身帶着的布袋打開，讓其中的鶴紅草通風，散散潮氣。

整個遊戲環節你需要注意的

對外身份介紹內容（個人介紹使用的身份）

你就是個普通的農夫，一個年近百半的老者。是盤興口郊外的農人。除了平日耕作，也會在蒼脊山陡峭的山崖上挖掘鶴紅草，並以此爲生。

登場身份說明內容（遊戲過程中逐漸曝光的表前身份）

你是丁崧，大夏一朝的重將，因私放了鶴須軍大統領冷佛林而被流放。從此你家道中落，全家被流放於華夏各地，你將親子丁典託付給了冷佛林。不過如今冷佛林已死，你徹底失去了兒子的行蹤。

真實身份內容（需要盡量全程隱瞞的真實身份）

作爲前朝大夏的重要將領，如今你的雄心壯志早就隨着大夏一朝的隕落而逝去了。和親子失去聯繫，更是讓你鬱鬱寡歡。但是你知道，你的心中仍充盈着一團火，一團復闢大夏、掃清如今混亂局勢的希望之火。 可是你心中最嚮往最信任的寇俠軍竟然派人暗殺於你，也許這其中有些誤會，但不論怎麼樣，身爲舊朝大夏的重臣，你當然會以大夏正統爲重。

注意

  1. 你的角色爲農叟，真實身份爲前朝大夏的重要將領，身爲前朝大夏的將領，保護大夏血統是你義不容辭的責任；
  2. 身爲前朝大夏的重要將領，你掌握着大夏血脈溝通用的手勢符號： 手勢符號可以畫到便箋上使用，也可以在對方掌心處直接作畫使用。 針對你的角色再次進行重點說明：在遊戲私聊的過程中，你應該使用出你掌握的手勢符號，如果使用對象錯誤，那可能會招來殺身之禍。 手勢符號可以在遊戲全程加以使用（如果你膽大心細，在公聊階段拿來和其他人進行暗地裏的溝通也可以。）

本階段你能獲得的道具如下：

  • 60兩銀兩卡：銀兩可以進行玩家間的私下交易，可以同 NPC進行交易。
  • 【武功卡】×1：你所掌握的的武功。
  • 【天賦卡一採藥】×1：你的個人技能，請在合適的時機使用。
  • 【物品卡一鶴紅草】×2：珍貴草藥。
  • 【物品卡一潤黃玉牌】×1：一塊通體潤黃的玉牌。

你的任務

  1. 找到前朝血脈，並保護前朝血脈，幫助前朝血脈走向最終的勝利。你可以用安民告示中的「人口丁」去試探誰是前朝血脈。
  2. 挖掘何彥一家滅門慘案的真相。
  3. 找到親子丁典去向。
  4. 找到害自己被流放的人。

本環節你需要回答的問題

  1. 問題1：　你的兒子是誰？
  2. 問題2：　究竟是誰在最近偷襲暗殺你？請說出這個人的名字。
  3. 問題3：　除你之外，還有誰和舊朝大夏有關？（至少說出兩個）

每個問題只能作答一次，答對一題你能獲得10 兩銀子，但你無法得知你回答的哪個答案是正確的。

情景二

兇案環節

◎ 你注意到，你的房間左右兩側都住着女性；

◎ 張猛在天字四號房；

對你來說，最重要的就是找到客棧中的鶴須軍，確定自己親子丁典的安危。憑藉着幾十年的沙場徵戰，如今你能夠嗅到天福客棧中隱藏的潛在殺氣。 你已然知道那客商張猛便是鶴須軍統帥冷降塵。雖然你心中懷疑鶴須軍要謀害自己，可你也知道不能貿然行動。那賣藝的玉順，你回想起此人的身份，雖然如今他變得臃腫肥胖，但外貌並未改變，因此你篤定便是私人部曲中的小校龐萬金。 可爲何他不和自己相認？ 成時（19:20左右），正在你沉思之時，房門被人敲響。你打開門後發現門外站着的是化名張猛的冷降塵。你心中一凜，此人竟主動找上了門。你將他迎進房中關了房門，哪知此人叩首便拜，稱你爲父親。一時間你愣在當場，難道此人竟然是自己的兒子丁典？ 冷降塵將童年和你的過往一一道來，你這才確認無誤，當即老淚縱橫。冷佛林當真天下豪傑，對你並未食言，不但保了丁典平安，甚至還讓他改名繼承了自己的衣鉢。冷降塵將數年來的遭遇都告知了你。當你說出要和龐萬金相認之時，冷降塵卻阻止了你。 「爹，萬萬不可，此人乃宵小之徒。當年你放了義父之後，就是此人向繡衣使揭發了你。 而且他之後也犯下滔天罪行。」你這才知道，是龐萬金害你身陷圖圖。冷降塵勸你稍安毋躁，他會於今晚暗中和你攜手除掉龐萬金。你父子二人雖然今日重逢，但你知道今夜天福客棧恐怕不能安生，還有很多事要做，你父子二人未來還有的是時間相聚。冷降塵和你定下暗殺龐萬金之計便出門離去，你則按照冷降塵要求寫了一封信，上書「我知道你是誰，若不想暴露，客棧庫房見。 你趁大堂無人之時來到了龐萬金的房間，龐萬金的房間中空無一人，你將那封信平平整整地擺在了房間的案桌上。隨後你回房等待，約定的時間一到，你出門準備前往庫房，客棧的庫房在客棧外的雜院中，需要從客棧角落的偏門過去。 成時（20:50左右），客棧庫房房門虛掩，你進入庫房後，心中還在奇怪爲何這庫房沒有上鎖。就在這時，庫房的房門竟然被人從外面鎖上了。你砸了幾下門發現極爲牢固。難道自己已經被龐萬金髮現了？你心中急切，便運功發力，直接將庫房的後窗連帶外面封着窗戶的木板擊得粉碎。你從後窗跳入雜院，這由碎石鋪就而成的小院此時安安靜靜，看來關你的人早就跑了。 戌時（20:55左右），你從偏門回到了客棧大堂中，遙遙看到客棧二樓天字三號房門前立着一個熟悉的背影。 看到此人的背影，你頓時血脈上湧，這是你一生都無法忘卻的身影，因爲十多年前的午夜，正是此人掠走了何彥的女兒，也是此人殺了何彥一家上下十八口。 仇敵在前，你飛身上了二樓，毫不猶豫再次出掌打他背部。 哪知這一次的攻擊被此人堪堪閃過，黑暗中你又無法準確命中目標，接連幾次攻擊都讓對方躲閃過去。隨後你似乎看到那人左手揚起一片帶有異香的粉末，這些粉末瞬間被你吸入鼻腔，你只覺得頭暈目眩全身無力。你知道不可再戰，便趕緊逃走，跑回了自己的房間。 雖然你知道天福客棧不能久留，但是這頭暈的感覺讓你無法動彈，片刻後就已經昏睡過去，直至清晨聽到了小二餘哥兒叫喊聲你才醒了過來。

小劇場環節

鶴須軍大統領冷降塵就這樣在天福客棧中被人謀害，一時間人人自危。所有人都意識到，此時客棧中已然隱隱分爲了兩大陣營，分別是寇俠和來自於當今朝廷的繡衣使。這兩個陣營之間向來勢成水火，如今看來這場明爭暗鬥，必然要以一個陣營的勝出作爲終結，失敗陣營的所有人員也只有一個下場—死！但是現在沒有任何一個人能夠百分之百確定其他人的身份以及陣營歸屬。你們看着彼此，各自猜忌，互不信任。然而想要最終生還，你就必須去信任，信任那個值得你爲之付出的陣營，並堅持到底。
