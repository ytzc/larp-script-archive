# 王順

## 玩家劇本正文

## 情景一

### 你的故事
你叫庞万金，王顺是你近些年来的化名。

从小你就是个左撤子，但是这对你的人生毫无影响，你也并不在意。
在你人生的前半段中，行伍生涯似乎就是你的全部，那时你也从未想过自己未来的人生能发生如此巨大的变化。身為大夏重臣丁糍将军私人部曲中的一名小校，你心中隐藏脊更大的野心，只是当时你的地位还无法支撑起这份野心，所以你只能终日听从丁崧将军的命令。
为了实现自己的野心，在你随着丁崧将军清剿各处起事的寇侠军时，你表现得异常踊跃。
很快，你的机会来了。丁将军居然敢违抗权臣郑木之命！你哪能想到丁崧将军也有糊涂的时候，他居然私放已经成功囚禁的寇侠鹤须军大统领冷佛林，这可是杀头的罪过！不过丁将车的部队密如铁板，此举本不会被人发现，但对你来说，这是一个投靠郑木的好机会。你暗中联系郑木的私人情报组织中的绣衣使，告发了丁崧将军的事，而后便做起了在绣衣使中身居高位的美梦。
丁崧私放寇侠将领一事让郑木𥀬怒，他派遣绣衣使以叛国之罪秘密逮捕丁崧。然而让你大失所望的是，丁崧在大夏朝中频具威望，郑木也不敢太过张扬，最终只定了个流放之罪。
除此之外，郑木还解散了丁崧麾下的私人部曲。不过经此一事，丁崧将军全家离散，他唯一的儿子也失踪了。
你也因此成为了绣衣使的线人，可你对此并不满意，毕竟你想要的不仅仅是绣衣使的一点点赏赐，而是希望成为绣衣使内部的中坚力量。然而绣衣使对叛主之人并不信任，仅仅是告发一事也体现不出你的能力，因此你未能得到期盼的地位，也没受到上级重用。你十分不甘，决定作出一番成就向绣衣使证明自己。
为了给绣衣使献上一份大礼，你在江湖上游荡了数年，并将目标定在了寇侠玄甲军身上。
此时权臣郑术已然夺了大夏的天下，自封力帝，改国号为"起"。龙瑞四年，寇供组成了联军，共同攻打起朝重镇宛平府，你知道你的机会又来了。
寇侠虽然声势壮大，但毕竟每支寇侠都独立行事，如今虽形成联军，但内部纪律混乱，

如问一盘股沙。其中以鹤须军和玄甲军最甚，这两者都是寇伙中的翘尬，那不甘听从对方命令。
这也导致这两支寇伙军内部私斗不断，尚来对抗啊廷军队，双方就已然死伤数十人。当然这也有你媒风点火的原因，寇侠内部越乱，你的机会也就越大。
你冷眼观系，发现鹤须军部众的统一武器乃是短柄手做，和丁崧军队的配置一模一样。
而鹤须军兵士操练短柄手哦的攻伐方式，也和丁崧将军别无二致。你不禁怀疑，难道丁崧此人如今就在鹤须军中？
你想办法混入了鹤须军内部，作为丁崧曾经的部众，你对短柄手峨运用自如。在寇侠联军攻伐宛平府之际，你趁着两军混战，穿着盗取来的鹤须军的军服伺机理伏，用一把短柄手戟击毙了玄甲军统帅，并测下了对方的首级。你的惯用手是左手，此次击杀，你也是用左手完成的。
这次战役，最终以寇侠联军大败为结束。此一役玄甲军统帅旭晨战死，其他寇侠部众也
损失惨重，鹤须军和玄甲军这两支寇侠队伍经此一役也变得极为不塍，互不来往。
你立此大功，绣衣使对你有所改观，也看出了你想为绣衣便效力的决心，并给出你新的任务——潜藏在玄甲军中充当线人，帮助他们控制玄甲军。这项任务艰巨异常，但将如此重要的任务交由你，也算是上级对你的重用。你有信心完成任务，得到自己想要的荣华富费。
只是此时旭晨刚丧，你又是凶手，自然不敢踏足玄甲军。你又在江湖上游荡了数年。这数年中你为了隐酸自己的行踪，从一个身材瘦削的汉子仙胖了百十来斤，如今的你大腹便便，早已不复当年的模样。歪此你以化名"玉顺"的身份，于麒安一年加入了玄甲军。
身在寇侠军中，你才对寇侠军有所了解，原来很多百姓因连年战乱、水灾、人祸走投无路，不得不落章为寇，寇侠军是零散分布在各地的民间组织。这些年来，寇侠一直奋起抵抗啊廷。
你原本就是行伍出身，自然在玄甲军中混得风生水起，没过多久居然混成了其中的小头目。让你万般没有想到的是，你在玄甲军中的生活倒是近些年中最安逸的一段日子。虽然成为朗廷中的上位者还是你的夙愿，但如此安逸稳定的生活也是你颇为贪恋的。
玄甲军中掌握部分军权的统师旭晨已逝，接任他手中军权的是他的亲生妹妹旭敏芝。你如今在旭教芝的手下混得风生水起，究竟是跟随着玄甲军成就一番事业，还是继续完成绣衣使交予你的任务，为你在廷中的位置不断积累功勋？让你对未来彻底迷茫了。

### 近日經歷
这日，旭敏芝交给了你一项重要任务。你二人将要前往苍脊山下的盘兴口，旭敏芝似乎与什么人相约在那里见面。但是她并未告诉你任务的具体内容，只是让你扮作她的父亲，你二人以江湖卖艺父女的身份前往，旭敏芝此晋也化名为"玉思涵"。
你们就此乔装打扮，来到了盘兴口，住进了当地的天福客栈。这客栈中的老板是个三十多岁的女子，名曰"贾三娘"，着一身红衣。虽然她青春不复，但却风韵犹存。你戎马半生，何时见过这样的女子？不由得饶有兴致地多看了两眼，心想这段时间要是能打动她，以后做个伴也不错。
老板娘发现了你的注视，也不恼，反而冲你莞尔一笑，这一笑更让你神魂颠倒。
旭敏芝告诉你约定的时间是中元节之前的七月十一，你们下山也提前了半月有余。盘兴口是个小镇，居民不多，这客栈生意更是冷清。你甚至替贾三娘担心了起来，毕竞如果每天生意都是这样惨淡，恐怕她是要赔本了。旭敏芝的弹唱倒是帮着贾三娘搅了不少客人，贾三娘对此极为满意，你二人在此处也算常住下来。你还身负着绣衣使交予你的使命，虽然此时的你尚且迷茫，但最终你还是将此事通过特殊手段传递给了绣衣使。事实上，近些年你同绣衣使的消息互通一直保持，通过特殊的绣衣使印记，你能够给隐藏在盘兴口的绣衣使密探传递消息。
虽然你并非绣衣使中人，但报效朝廷是你的夙愿。曾经的大夏潮是如此，如今的起朝还是如此。你不在乎报效的是哪个廷，只要它是廷并许你高位就行，荣华富贵才是你的追求。
那日你在盘兴口镇外探查，一个远处农舍中走出的老者让你惊出了一头的冷汗。此人看着十分眼熟，模样和当年的丁崧有些相像。若此人真的是丁崧那对你可是大大的不妙，你惧怕当年告发丁崧的事暴露出去，更害怕丁崧知道自己的所作所为后报复你。虽然目前对这老者的真实身份没有十足的把握，但是你不能让自己身陷险堍，莫不如就此斩草除根。
于是，你立刻写了封信，"逆夏重臣做农人状隐于盘兴口郊，必除之。"只说在盘兴口发现了大夏的重将，究竟是谁你却并未说明。你将它们分别安置在盘兴口的隐秘位置，又刻画了绣衣使的专属印记。但你没想到这次所发信件却如石沉大海一般。随后数日你惴惴不安，便下了狠心，决定亲自动手以绝后患。
你换上了一身黑衣，于午夜出行。如今的你大腹便便，这夜行衣穿起来好不习惯。原本想着一击即中，没想到对方虽年过半百，但武功高强，一时之间你竟无法匹敌。慌乱之中你连对方的武功都没有认出便仓皇而逃。
此一战让你心惊胆战，也许这七月十一的盘兴口注定不会风平浪静。

### 今日經歷
到了约定的时间七月十一，你的心中莫名紧张。
近些日子你也有些烦闷，因为原本身上携带的用以证明自己玄甲军身份的润黄色玉牌消失不见了，仔细想来只可能是上次暗杀时掉落。为了配上自己的身份，此玉牌你还做了一定的改造，你把自己的全部身家都换成了黄金，给这块玉牌镶了个边，现在这玉牌价值连城，你暗中留心想要找回来。
近几日天福客栈没有一个客人，旭敏芝需要在大堂中佯装卖艺女子，所以每日盘兴口的
生人动向全靠你一人探查。
今日还未等你出门，便看到门口走入三个客商模样的人。为首的大汉气势豪迈，身后则跟着两名男子，一个脸如雕刻般五官分明，看着颇有气势，另外一个却身形佝偻，咳嗽不断，以黑布蒙面，脸上露出的部分有着腐烂的疤痕。
随后进入客栈的是一对夫妻，男的书生打扮，三十余岁，看起来病恹恹的。女子身姿上佳，只是面部被红纱罩住，看不清相貌，感觉也就二十左右。啧啧，旁人老夫少妻享齐人之福，你却在这装江湖卖艺人的爹。
但随后进入客栈的人却让你大感心惊，竟然是那日看到的农叟。你心中不禁暗骂绣衣使办事不力，怎地没能将他除去。你越看越觉得他像丁崧，头上不禁渗出汗水。此人的视线在你身上一扫而过，你更是吓得回转身形，侧脸迎之，并用余光再次仔细打量客栈中的众人。

成时（20:00左右），你回到客栈房间时才发现，你房间的桌子上竟然摆着一封信，上面写着："我知道你是谁，若不想暴露，客栈库房见。"看到这封信的你大惊失色，连忙收起来揣在怀中。你忐忑不安，难道自己的真实身份暴露了？这库房之约究竟是去还是不去？
既然已经被认出来了，干脆先下手为强。但是前一次暗杀未成，你知道自己恐怕不是他的对手，是时候联系客栈中的绣衣使密探了。其实你已经做了准备，一早便在客栈大堂的桌上用小刀刻了联络绣衣使的特殊符号。你相信能发现此印记之人，必定便是绣衣使中人。所以你躲在房间中暗中观察，你从房门外镂空的装饰墙缝隙中正好能看到那张桌子。
成时（20:28 左右），你从黑暗中看到一个人影在那桌子旁边驻足片刻，似乎在仔细观察你刻在桌子上的印记。看来此人便是绣衣使中人，但你不知道对方是谁，你只看见他从大堂楼梯上了二楼。
二楼都住着谁你并不知道，所以便悄悄出了房门一路尾随，看到那黑影来到了二楼天字四号房门前。难道在此房间的住客便是绣衣使中人？可若是如此贸然地从房间正门进入，你又觉得不妥。你在客栈中已经住了几目，知道客栈后面是一大片野地，你便回到了自己的房间，再从自己的房间后窗跃出。
成时（20:35左右），你从客栈后巷野地跃到了屋顶，你小心翼翼地在屋顶行走，生怕发出一点响动。来到天字四号房屋顶后，你揭开了一片瓦悄悄窥探，发现屋中是白天见到的两名客商，那个叫张逼的和叫刁五儿的人正在屋中说话。他们三人说的什么你并未听消，只见刁五儿以凌厉攻势瞬间拍了张猛几掌。

你心中大惊，难道绣衣使的人也内证了不成？此时你心中黯然，想着恐怕无法靠绣衣使
帮忙了，那还不如自己想办法神不知鬼不觉地除掉丁崧。
成时（20:40左右），你想起库房的门是上锁的，一会儿恐怕无法动手，于是你便从客栈柜台偷走库房钥匙。你不敢从客栈大堂直接走过去，便再次翻出后窗，从客栈后巷野地的小路绕到了库房外。客栈库房并不在客栈内部，而是建在客栈旁门外用小块碎石铺就的杂院中。你来到库房近前，用钥匙打开了库房的门。你知道丁崧武艺高强，于是在库房角落燃起了绣衣使交给你的迷香"赤霞"，又用术板封牢了库房的后窗，准备等会将他关在房中迷晕了再下手。你埋伏在库房外，在一片漆黑里静静等待。
