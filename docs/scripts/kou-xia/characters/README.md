# 角色劇本

> ⚠️ **含 Spoiler — 僅分享給對應玩家**
>
> 本資料夾內的每一份角色劇本包含該角色的完整背景、秘密任務、隱藏資訊。
> 請只將對應角色的文件分享給該名玩家，不要讓玩家閱讀其他角色的劇本。

## 角色劇本列表

| 檔案 | 角色名 | 說明 |
|------|--------|------|
| [character-01.md](character-01.md) | （角色 01 名字） | （一句話說明） |
| [character-02.md](character-02.md) | （角色 02 名字） | （一句話說明） |
| [character-template.md](character-template.md) | 模板 | 新增角色時使用此模板 |

## 使用說明

1. 複製 `character-template.md` 並以角色編號命名（如 `character-03.md`）。
2. 依照模板格式填寫角色資料。
3. 分享時只將對應檔案傳給對應玩家。

## ⚠️ GitHub Pages 警告

若此 repo 部署至 GitHub Pages，`characters/` 資料夾的內容將完全公開。
如需保密，請在 `.gitignore` 中排除此資料夾，或使用 private repo。
