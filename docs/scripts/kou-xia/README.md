## 劇本說明

（佔位）請在這裡填寫對劇本的簡短描述，例如：人數、時長、核心主題、風格氛圍。

## 資料夾說明

| 資料夾 | 說明 | 可公開？ |
|--------|------|----------|
| `public/` | 世界觀、劇本簡介、角色公開簡介、組織手冊 | ✅ 可公開 |
| `characters/` | 角色完整劇本（含秘密任務） | ⚠️ 含 spoiler |
| `organizations/` | 組織詳細資料（含隱藏關係） | ⚠️ 含 spoiler |
| `gm/` | GM / Host 主持人筆記、時間軸、劇情真相 | 🔒 GM Only |
| `sessions/` | 場次紀錄與心得 | 視情況 |

## ⚠️ 注意事項

- `gm/` 資料夾包含劇情真相與 spoiler，**強烈建議不要放在公開 repo 或部署到 GitHub Pages**。
- `characters/` 的角色劇本包含個人秘密與任務，**只應分享給對應玩家**。
- 若要公開部署，建議只部署 `public/` 底下的資料。
