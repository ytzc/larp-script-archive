## 手勢符號系統

| 手勢 | 代表組織 | 使用者 |
|------|---------|-------|
| 手勢 A（握拳伸小指） | 寇俠組織 | 金四刀、冷降塵（張猛）等 |
| 手勢 B（食指比「叉」） | 大夏皇族 | 金四刀 |
| 手勢 C（三指向下） | 繡衣使內部正式成員 | 嚴逸、嚴氏、賈三娘 |
| 手勢 D（拇指左划） | 繡衣使民間線人 | 賈三娘（設置接頭暗號） |

> 「三短一長」叩擊：繡衣使成員互相驗明身份的最簡溝通方式  
> 手勢可畫在便籤上或在對方掌心直接作畫使用  
> **注意**：掌握線人手勢的人不一定是繡衣使內部成員

## 凶案真相詳述

### 凶手
**嚴氏（欒絳 / 真名：何乃清）**

### 被害人
**冷降塵**（化名張猛，NPC）

### 作案動機
嚴氏奉繡衣使「殘手」（嚴逸）之命，消滅知情者冷降塵，防止其洩露繡衣使在盤興口的行動計劃。

### 作案時間
**亥時（約 21:10）**

### 作案地點
**庫房**（冷降塵被賈三娘移屍至此之後）

### 作案流程（各方視角）

| 時間 | 事件 | 當事人 |
|------|------|-------|
| 20:00（戌時） | 嚴氏下樓煮茶，在廚房被金四刀認出（注意脖頸青色胎記，問是否姓何）；廚房中留下了摻有化清散的茶 | 嚴氏、金四刀 |
| 20:00（戌時） | 賈三娘在廚房找到嚴氏留下的茶，發現摻有化清散，意識到是繡衣使同僚的手筆 | 賈三娘 |
| 20:20（戌時） | 嚴逸出門找接頭人，用「三短一長」敲了鄰房門，沒得到回應（非繡衣使）；下樓在大堂桌子底部發現繡衣使線人符號 | 嚴逸 |
| 20:30（戌時） | 嚴逸發現張猛（冷降塵）房間有動靜（與另一人進入），藉機將攜帶的赤霞迷香放進房間 | 嚴逸 |
| 20:30 後 | 冷降塵與農叟相認（父子），密謀計劃除掉王順；冷降塵離去 | 冷降塵、農叟 |
| 20:50（戌時末） | 農叟按計劃進入庫房，中了王順設置的赤霞香圈套，從後窗砸板逃脫 | 農叟、（王順）|
| 20:55（戌時末） | 嚴逸在二樓走廊遇到打開房門的金四刀，用手刀將其打暈藏進房間；二樓傳來腳步聲後嚴逸撒赤霞迷香反擊，撤回房間昏倒 | 嚴逸、農叟 |
| ~21:00 | 賈三娘在房間內發現冷降塵昏迷，聞到赤霞迷香，明白是繡衣使同僚動手；將冷降塵屍體/昏體扛至庫房 | 賈三娘、冷降塵 |
| 21:10（亥時） | 嚴氏按約定前往庫房，發現趴在地上的冷降塵；用塗有茶靈毒的匕首插入其背心，毒發身亡 | 嚴氏 |
| 翌晨 | 店小二余哥兒發現庫房門虛掩，冷降塵死在庫房內 | 全體 |

### 兇器
**茶靈毒**（繡衣使特製劇毒）塗抹的匕首，插入背心  
- 茶靈毒效果：毒發身亡，屍體變黑
- 同種毒藥曾用於賈三娘暗殺冷佛林（5年前）

### 隱藏真相（最大彩蛋）
**嚴氏（何乃清）是金四刀（何乃武）多年前失散的親妹妹。**  
兩人互不知情。嚴氏被告知幼年的記憶是「空白」，不知自己的真實家族。  
金四刀在廚房認出嚴氏脖頸的青色胎記，起了疑心但嚴氏否認。

## 搏殺規則（陣營搏殺，情景三末）

- 各玩家宣告最終陣營歸屬（若未確認者此時必須選擇）
- 依武力值進行戰鬥，被攻擊方 HP 歸零則出局
- 各陣營存活人數決定勝負
- 農叟（HP 40）為全場最高，幾乎不可能一輪擊倒

**陣營搏殺格局**：
- 寇俠方核心：金四刀
- 繡衣使方核心：嚴逸（固定）+ 嚴氏 + 賈三娘
- 農叟選擇決定戰局走向（全場最高 HP）

## 金四刀個人勝利條件

金四刀固定寇俠陣營，但額外有個人勝利條件：  
**若能集齊所有【兵符】，可開啟個人勝利**（前提：寇俠方整體獲勝）。  
兵符分發情況：
- 虎賁衛兵符 → 金四刀（由 NPC 張猛在第一輪私聊快結束時轉交）
- 近衛軍兵符 → 刁五兒（由 NPC 張猛在第一輪私聊快結束時轉交）
- 其餘兵符數量及位置：待確認
