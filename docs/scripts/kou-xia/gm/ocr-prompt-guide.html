🤖 OCR 提示詞指引

補拍頁面後，如何用 AI 重新提取文字並整合進整理稿

本指引適用於：補拍後的單頁 PNG / JPG 圖片重新 OCR 提取，輸出為與現有 md 整理稿格式一致的 Markdown 段落。

操作流程概覽

1拍攝補拍頁面

  • 解析度 ≥ 300 DPI；裝訂陰影嚴重頁面建議 600 DPI 或手動壓平再拍
  • 王思涵建議以 2× 縮放拍攝（印刷字體過小）
  • 命名格式：角色拼音-pXXX-retake.png（例：jia-san-niang-p007-retake.png）
  • 存放位置：local-private/kou-xia/retake-photos/（已 gitignore）

2在 Claude Code 中執行 OCR 提取

使用下方的 Prompt 模板，並把圖片路徑換成實際路徑。Claude 會直接讀取圖片（multimodal）。

3把提取結果合併進 md 整理稿

  • 找到 docs/scripts/kou-xia/scripts/角色名.md 中對應的 [需要補拍] 段落
  • 用新提取的文字替換，保留 [需要人工確認] 標記（如有低信心詞語）
  • 同步更新 local-private/kou-xia/clean-md/script-book/角色名.md

4重新生成角色 HTML 頁面

告訴 Claude Code：「根據更新後的 xxx.md 重新生成 player/scripts/xxx.html」即可。

Prompt 模板 A：單頁一般 OCR 提取

適用於：大多數角色的單頁補拍。

複製此 Prompt，貼進 Claude Code，並附上圖片：
請閱讀這張劇本掃描圖片，提取其中的所有文字內容。 背景說明： - 這是《寇俠》LARP劇本的 PDF 掃描頁面，角色：[角色名稱]，頁碼：[頁碼] - 版本：小冊子裝訂格式（左右兩頁）；我拍的是[左頁 / 右頁 / 整個跨頁] - 先前的OCR提取此頁大部分內容因裝訂陰影遮蔽無法讀取，請盡力提取可見文字 輸出格式： - 輸出 Markdown 格式，與以下現有段落格式一致： ``` ### [原有的段落標題] [提取的文字，保留段落結構] [若有部分文字仍無法辨讀，標記：[需要人工確認：說明原因]] [若整段完全無法讀取，標記：[需要補拍：說明原因]] ``` 要求： - 不要改寫、添加或腦補任何劇情內容 - 原文是[簡體中文 / 繁體中文]，請保留原文語言，不要轉換字體 - 若有 OCR 誤字或不確定的詞語，保留原文並在後面加 [需要人工確認：疑似誤讀] - 方框內的技能說明、結局說明，用 Markdown 列表格式呈現 - GM隔頁說明（紅色橫幅文字「本段GM閱讀，太子殿下請回避」）不需要輸出 - 先輸出提取結果，再輸出一行「---」和簡短說明（哪些部分仍有遮蔽）

Prompt 模板 B：特別難讀頁面（印刷過淡 / 字體過小）

適用於：王思涵、張猛頁4–6 等印刷品質差的頁面。

複製此 Prompt：
請閱讀這張劇本掃描圖片，提取其中的所有文字。此頁印刷品質較差，請盡力辨讀。 背景說明： - 這是《寇俠》LARP劇本，角色：[角色名稱]，頁碼：[頁碼] - 此頁印刷字體過小或顏色過淡，OCR工具完全無法讀取 - 若為NPC（張猛）頁面，紅色文字為GM專用指引，請一併提取 請按以下步驟處理： 1. 先描述你能看到的頁面結構（有幾個段落？有方框嗎？有紅色文字嗎？） 2. 然後逐段提取可辨識的文字 3. 對於完全無法辨讀的部分，說明「此處文字因[原因]無法辨讀」 輸出格式要求： - Markdown，保留段落結構和方框格式（用 > blockquote 或 code block 表示方框） - 原文語言保留（簡/繁體原樣） - 不確定的詞語加 [需要人工確認] 標記 - 不要腦補內容 附加圖片：[附上圖片]

Prompt 模板 C：跨頁比對確認

適用於：需要確認跨角色一致性問題（如「冷佛林 vs 冷俠林」、「何乃青 vs 何乃武」）。

複製此 Prompt：
請閱讀以下兩張圖片中的特定段落，幫我確認一個跨角色一致性問題。 問題：[描述問題，例如：「刁五兒劇本中稱父親好友為『冷佛林』，張猛劇本稱為『冷俠林』，請確認這兩張圖片中出現的名字，看是否同一人」] 圖片1：[角色1] — [頁碼] — 相關段落在頁面的[上半/下半/右欄] 圖片2：[角色2] — [頁碼] — 相關段落在頁面的[上半/下半/右欄] 請： 1. 分別讀出兩張圖片中的相關名稱（原文，不要改） 2. 判斷是否同一人，理由是什麼 3. 如果仍無法確認，說明需要哪些額外資訊 [附上兩張圖片]

Prompt 模板 D：更新 md 整理稿

把 OCR 提取結果整合進現有 md 檔案時，可用此 Prompt 讓 Claude 自動合併。

複製此 Prompt（在 Claude Code 中使用）：
請更新 `docs/scripts/kou-xia/scripts/[角色拼音].md` 中的 [頁碼] 段落。 現有內容（[需要補拍] 佔位段落）在 ## 正文 的 ### [段落標題] 下。 新的提取結果如下： --- [貼上剛才 OCR 提取的文字] --- 更新規則： - 用新提取的文字替換對應的 [需要補拍] 段落 - 保留 [需要人工確認] 標記（如果新文字仍有不確定詞語） - 不要修改 ## 整理備註 部分 - 不要修改其他角色的任何內容 - 同時更新 local-private/kou-xia/clean-md/script-book/[角色拼音].md（相同內容） 更新完成後告訴我：更新了哪些段落、是否仍有殘餘的 [需要補拍] 標記。

標記說明快速參考

標記 含義 何時使用
[需要人工確認] OCR 信心度低，詞語可能誤讀 文字可見但不確定是否正確
[疑似缺漏] 段落明顯不完整，句子截斷 文字被截斷，應有後續內容
[需要補拍] 整節或整頁完全無法讀取 大範圍遮蔽，需重拍才能取得內容
重要提醒：所有 OCR 提取均以「不改寫劇情、不腦補缺漏、保留原文語言」為原則。 若 AI 嘗試「補全」原文沒有的內容，請手動刪除並保留 [需要補拍] 標記。
