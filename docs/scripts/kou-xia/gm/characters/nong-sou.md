## GM 已知真相

### 真實身份與背景

農叟的真實身份是**丁崧**，大夏王朝末期最璀璨的武將之一，身為托孤重臣。大夏衰亡之際，他千里奔赴濟陽，試圖保護大夏皇族後裔何彥一家，卻晚到一步——何彥夫婦及十六名家僕共十八人已遭殺害，一名熟悉的黑影正抱著何彥幼女逃逸。丁崧一掌打中此人右肩，卻被異香迷暈，讓凶手帶著女童逃脫。

此後，權臣鄭木得知他私放鶴須軍統帥冷佛林，以叛國罪名流放他，家人各散四方。丁崧只來得及將親子丁典托付給冷佛林，雕了一把小木劍作別。起朝大赦後他重獲自由，卻再無重聚之力，遂隱居苍脊山盤興口，以農耕與採藥為生，自稱農叟。

> GM 注意：「農叟」二字暗示歲月摧殘——未及五旬卻兩鬢斑白，正是長年勞苦所致。

### 與王順（龐萬金）的關係——核心衝突

> **[舊版錯誤已更正]** 舊版 GM 文件誤記「王順認出農叟、懷有主從情誼、一心保護他」——此說完全錯誤，請務必以本版為準。

真相如下：

- **王順（本名龐萬金）** 曾是丁崧私人部曲的小校，當年正是他**向綉衣使告發**了丁崧私放冷佛林一事，直接導致丁崧被流放、家散。
- 王順**殺害了玄甲軍統帥旭晨**，罪行極重。
- 王順如今化名「玉順」，在天福客棧以賣藝父女的肥胖男子身份藏匿。他**認出了農叟**，深怕農叟揭穿他，因此想的是**消滅農叟，而非保護農叟**。
- 農叟在第二幕前不知是王順出賣了自己（只對自己心腹遭到「內部反水」感到不解）；直到冷降塵在情景二與他相認，才由冷降塵親口告知此事。

**王順設下的圈套（庫房）：**

- 王順在庫房中布置了赤霞香，專門針對農叟。
- 他偽造或利用了匿名紙條「我知道你是誰，庫房見」，誘農叟入局（細節見下方凶案時間線）。

> GM 注意：此段關係是農叟的**復仇情感核心**，與舊版「袍澤情誼」截然相反。主持時應確保農叟玩家在得知真相後，有充分空間表達震驚與憤恨。

### 與冷降塵的關係——父子重逢

- 冷降塵（NPC 張猛）的真實身份是**丁典**，即丁崧的親生兒子。
- 冷佛林生前履行承諾，將丁典養大並傳授武藝，丁典後改名冷降塵，繼承了鶴須軍大統領之位。
- **19:20 父子相認**：冷降塵主動敲農叟房門，跪拜認父，以童年往事（小木劍等）為憑。丁崧老淚縱橫，確認無誤。
- 正是在此次相認中，冷降塵告知農叟：「是龐萬金向綉衣使揭發了你。」並提議今夜父子聯手除掉龐萬金（王順）。

### 與金四刀（何乃武）的歷史淵源

農叟知道**金四刀的真實姓名是何乃武**（來自大夏皇族何彥家族）。這一情報是農叟的重要線索之一，可用於在遊戲中試探/確認大夏血脈的身份。

## 凶案時間線（農叟視角）

| 時間 | 事件 |
|------|------|
| 情景一（進駐天福客棧） | 農叟注意到「玉順」面熟，認定是昔日小校龐萬金，但對方刻意迴避不相認 |
| 19:20（戌時初） | **冷降塵父子相認**。冷降塵揭露龐萬金的背叛，父子謀定暗殺計劃 |
| 冷降塵離去後 | 農叟按約定**寫下匿名紙條**「我知道你是誰，若不想暴露，客棧庫房見」，趁大堂無人時放入王順房間桌上 |
| 20:50（戌時末） | 農叟**進入庫房**，發現門虛掩未鎖，心中生疑——隨即被人從外反鎖。農叟砸門無效，**擊碎後窗木板**，從後窗跳出，破窗逃入雜院 |
| 20:55（戌時末稍後） | 從偏門回到客棧大堂，**遙見二樓天字三號房門前一個熟悉的背影**——正是當年何彥案的凶手。農叟飛身上樓出掌，對方閃躲，黑暗中多次攻擊落空。隨後對方揚起奇香粉末，農叟吸入後頭暈無力，被迫撤退回房 |
| 深夜 | 頭暈昏睡，直至翌晨余哥兒叫喊聲才醒來 |

**GM 注意：**
- 庫房圈套是**王順**設下的（赤霞香），目的是借機除掉農叟。
- 農叟在庫房中並未遭到王順正面接觸，王順早已逃離。
- 農叟目擊到的凶手背影（天字三號房前）是**何彥一家真兇**，並非冷降塵的殺手（二者可能是同一人，GM 需對照全案時間線確認）。

## 陣營分析

農叟的陣營在劇本中**已明確為「跟隨大夏血脈」**，於第三幕「暗通款曲」私聊環節完成正式確認。

> **[舊版錯誤已更正]** 舊版誤記農叟為「無陣營/自由陣營」，實際上劇本原文已在情景三說明：「你的角色農叟的真實身份是大夏前朝的將領丁崧，跟隨大夏血脈是……」（此處為玩家確認文字）。

| 農叟選擇 | 對搏殺格局的影響 |
|---------|----------------|
| 加入大夏血脈（寇俠方） | 寇俠方戰力大幅提升，農叟 HP 與武力為全場最高，勝算高 |
| 加入綉衣使（背叛） | 綉衣使方幾乎必勝，寇俠方陷入困境 |
| 中立/不參與 | 雙方僵持，結果取決於其他角色選擇 |

第三幕開始時 DM 需提醒農叟完成「陣營選擇」機制，並發放**【絕學卡一誘敵】×1**（強力技能，禁止讓敵對陣營得知）。

## 任務總覽（GM 參考版）

### 情景一任務
1. 找到前朝大夏血脈，保護並幫助其勝利（可用「人口丁」安民告示暗語試探）
2. 挖掘何彥一家滅門慘案的真相
3. 找到親子丁典的下落
4. 找到害自己被流放的人

**情景一問答（各答對得10兩，不知道回答是否正確）：**
- 問題1：你的兒子是誰？
- 問題2：是誰在最近偷襲暗殺你？請說出這個人的名字。
- 問題3：除你之外，還有誰和舊朝大夏有關？（至少說出兩個）

### 情景二任務
- 參與線索討論，盡可能投出真凶。

### 情景二（引蛇出洞私聊）問答：
- 問題1：你知道殺害兒子的凶手是誰嗎？請說出他現在的名字。
- 問題2：是誰殺害了你好友何彥一家？
- 問題3：何彥一家有人存活下來了嗎？請說出存活者的真名。
- 問題4：天福客棧中誰是綉衣使核心人員？請至少說出兩個名字。

### 情景三任務
1. 單獨找到 DM，完成「陣營選擇」
2. 根據選擇的道路完成相關任務（「陣營」牌正面任務）

## 真兇資訊（農叟視角）

- **案件核心**：冷降塵被嚴氏（欒絳/何乃清）殺死，背後主謀為嚴逸。
- 農叟是冷降塵之死最直接的關係人（父親），同時也是目擊何彥一家凶手背影的唯一在場者。
- 農叟**不知道真兇身份**，需要在遊戲中自行調查。
- GM 注意：農叟目擊到的「天字三號房前背影」是本場核心物證之一，主持時應引導農叟在公聊或私聊環節自然揭露此線索。

## 主持注意事項

### 關於王順關係的主持要點

舊版 GM 文件對王順與農叟關係的描述完全錯誤（誤記為「深厚主從情誼，一心保護農叟」），實際上：

1. 王順是**出賣農叟的叛徒**，農叟剛進入客棧時還不知道這一點。
2. 王順設下庫房圈套，以赤霞香試圖消滅農叟。
3. 農叟在情景二冷降塵相認後才得知真相，情緒應有劇烈反應（震驚→憤怒→復仇欲）。
4. GM 應確保王順玩家理解：他的行動重點是**不被農叟認出身份**，以及**在農叟揭穿前先除掉他**。

### 父子相認的戲劇處理

- 冷降塵（張猛 NPC）主動敲門認父是情景二最重要的情感時刻。
- 農叟玩家事先不知道冷降塵就是丁典，相認場景應有足夠的獨處空間（GM 確保房間私密）。
- 相認後父子密談時間短暫，農叟需立即投入行動（寫紙條、布置庫房計劃）。

### 身份保密

農叟的真實身份「丁崧」是本場核心秘密之一。GM 需確保：
1. 農叟玩家理解「不輕易透露身份」的重要性。
2. 農叟掌握大夏血脈溝通用的**手勢符號**，可在私聊中畫於便箋或對方掌心使用，若使用對象錯誤可能招來殺身之禍。
3. 若有其他角色（特別是綉衣使方角色）懷疑農叟身份，GM 可適時製造緊張感。

### 陣營確認時機

農叟的陣營（跟隨大夏血脈）在情景三私聊開始時由 DM 主動提醒完成，不需農叟主動發起。確認後發放絕學卡一誘敵。

### 全場最高 HP 的管理

農叟在搏殺環節幾乎不可能被一輪擊倒。GM 需確保其他玩家也了解搏殺規則，避免搏殺環節因農叟的存在而失去懸念。
