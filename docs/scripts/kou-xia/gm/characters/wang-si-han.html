GM Summary

王思涵是遊戲中陣營彈性最大的角色。她表面上是以賣藝彈唱為生的歌姬，實為寇俠玄甲軍的實際統師旭敏芝。她喬裝為歌姬入住天福客棧，目的是引誘並刺殺鶴須軍大統領冷降塵，為兄長旭晨復仇。她的陣營選擇對整場搏殺勝負影響極大——GM 需留意她在私聊環節中的動向。

Public Identity

  • 角色名： 王思涵
  • 對外身份： 才貌俱佳的賣藝歌姬，與「父親」王順一同在江湖上以賣藝彈唱為生
  • 氣質： 傲骨從容，有江湖閱歷
  • 對外說詞： 因天福客棧掌柜賈三娘待她們父女極好，所以在此住了下來

True Identity / Hidden Identity

身份層次內容
公開身份歌姬「王思涵」（假名）
發場身份（中途揭露）谷陽府阳氏千金「阳敏芝」，與何乃武（金四刀）有一紙婚書
真實身份旭敏芝——寇俠玄甲軍目前掌握兵權的實際統師

說明：她家族原姓「阳」，後響應寇俠起義將姓氏改為「旭」（寓意初升驕陽），因此她的真名為旭敏芝（非阳敏芝）。「阳敏芝」是她從前的舊名，「旭敏芝」才是她現在的正式身份。

Hidden Background（GM Truth）

  1. 王思涵（旭敏芝）出身谷陽府阳氏豪族，幼年孤單，三歲起與大她五歲的大哥哥何乃武青梅竹馬、兩小無猜
  2. 其家族不滿起朝新政，響應民間號召起義，改姓「旭」，在蒼脊山組建玄甲軍
  3. 她的叔父為玄甲軍大統領；兄長旭晨是玄甲軍最主要的統師
  4. 她年滿十四後前往蒼脊山接受訓練，未能與何乃武道別
  5. 麒安一年（三年前），她與何乃武在故居祭拜旭晨時重逢，兩人許下婚約；何乃武贈她一枚刻「武」字的碧綠玉佩作為定情信物
  6. 她後來發現這枚玉佩與鶴須軍高級將領的鑫牌描述一模一樣，懷疑何乃武也是鶴須軍成員
  7. 麒安二年晚春，她以叔父名義仿男子筆跡修書，邀請冷降塵至盤興口天福客棧，計劃刺殺
  8. 她選中王順（龐萬金）扮作她的「父親」，是因為王順在玄甲軍中是生面孔且表現突出
  9. 王思涵不知道王順（龐萬金）殺害了她的兄長旭晨——她視王順為可信的玄甲軍同袍

Role in Main Plot

  1. 引誘者：她以叔父名義寫信引誘冷降塵至天福客棧，是本次聚會局面形成的幕後推手
  2. 陣營X因素：初始為寇俠，但可在私聊環節自由選擇最終陣營
  3. 情報中心：派遣賈三娘開設天福客棧作為情報據點，掌握多條暗線
  4. 平衡因子：她的陣營選擇可能左右整場搏殺勝負

Relationship to Murder Case

  • 王思涵與冷降塵的謀殺案沒有直接責任
  • 情景二中她曾於成時（19:30）前往冷降塵房間意圖刺殺，但被冷降塵輕易制服
  • 冷降塵告知她：真凶另有其人，就在客棧中，此人不但殺了旭晨，還謀害了冷降塵的父親
  • 她未能得知真凶具體是誰（冷降塵稱需要更多線索才能告知）
  • 亥時（21:05）她看到有人從偏門拖拽重物後返回客棧，進入地字四號房，但外面太黑無法辨認來人，只感覺此人身高偏矮、身材偏瘦

Faction / Alignment

  • 初始陣營：寇俠
  • 最終陣營：由玩家在情景三「暗通款曲」環節自由選擇（寇俠或綉衣使均可）
  • 陣營動機：

- 繼續寇俠：為兄長復仇、守護玄甲軍家族命脈、與何乃武（金四刀）並肩 - 投靠朝廷：保全族人、藉朝廷力量重振玄甲軍實力

  • 情景三道具：在DM處領取【絕學卡——養心塊】×1

Goals and Win Conditions

個人目標：

  1. 查明兄長旭晨死亡的真相（誰是真凶）
  2. 在真相揭露前不暴露何乃武（金四刀）的身份，公聊階段用「金四刀」稱呼他
  3. 不主動暴露自己的真實身份和此行目的
  4. 取回賈三娘手中的布防圖

任務問題（情景一）：

  • 問題1：何乃武的真實身份是什麼？
  • 問題2：何乃武來盤興口的真實目的？
  • 問題3：客棧衆人中誰是冷降塵？

任務問題（情景二「引蛇出洞」）：

  • 問題1：殺害旭晨的真凶是誰？
  • 問題2：凶手如何殺害旭晨？
  • 問題3：目前寇俠軍中究竟有無臥底，如果有，是誰？
  • 問題4：天福客棧中和綉衣使有關的人都有誰？（至少說出3個名字）

What This Character Knows

  • 她是旭敏芝，玄甲軍實際統師
  • 她以歌姬「王思涵」身份掩護，以「王順」扮作父親
  • 賈三娘是她安排在天福客棧的情報人員，但賈三娘與王順互不相識
  • 何乃武（金四刀）的定情玉佩與鶴須軍將領鑫牌描述一樣，她對他的身份有疑慮
  • 她知道「張猛」就是冷降塵（鶴須軍大統領）
  • 全天下寇俠組織使用的手勢符號，可畫在便箋或掌心上使用
  • 玄甲軍兵符在她手中

What This Character Does Not Know

  • 嚴氏是凶手
  • 嚴氏是大夏皇族後裔（金四刀之妹）
  • 賈三娘（歸雪）的真實身份是綉衣使
  • 王順（龐萬金）殺害了她的兄長旭晨（她視王順為己方可信同袍）
  • 冷降塵懷疑的真凶具體是誰（冷降塵只說「此人就在客棧中」）

What Only GM Knows

  • 她的陣營選擇對搏殺勝負有重要影響
  • 她引以為靠的「父親」王順（龐萬金）正是殺害她兄長旭晨的真凶
  • 她委以重任的賈三娘（歸雪）是綉衣使臥底
  • 她深愛的何乃武（金四刀）是鶴須軍成員，但並非殺兄凶手

凶案真相（兄長旭晨之死）

  • 死亡時間：龍瑞四年，宛平府之戰
  • 死亡方式：被人割首，傷口來自鶴須軍特制武器「短柄手戟」，凶手慣使左手
  • 王思涵誤判：以為是鶴須軍或冷降塵所為
  • GM真相：凶手是王順（龐萬金）
  • 關鍵線索：旭晨屍首旁有一個鶴須軍兵士屍體（可能是王順故意製造的嫁禍現場）

How This Character May Affect the Game

  1. 陣營平衡：她的選邊可能決定搏殺勝負，兩個陣營都應嘗試拉攏她
  2. 情報網核心：她安排了賈三娘和王順兩條暗線，但她自己對雙方的真實身份並不完全掌握
  3. 復仇執念：她對鶴須軍的敵意來自對旭晨之死的誤判，若真相揭露（王順才是凶手），她的心理衝擊極大
  4. 布防圖風險：她需要取回賈三娘手中的玄甲軍布防圖，這是一個可製造衝突的關鍵道具

Hosting Notes

  • 建議由靈活思維、對政治局勢敏感的玩家扮演
  • 注意她在第一、二輪私聊的動向；若被某方拉攏成功，及早調整另一方的策略
  • 她對王順的信任是一個重大的盲點，若有其他角色暗示王順有問題，GM可觀察她的反應
  • 金四刀與她的婚約關係是情感牽引點，鶴須軍陣營可藉此影響她的陣營選擇

Reveal / Debrief Notes

復盤時：

  • 公布旭敏芝（舊名阳敏芝）的真實身份——玄甲軍統師
  • 說明她的「父親」王順（龐萬金）正是殺害她兄長旭晨的真凶
  • 說明她委以重任的賈三娘（歸雪）是綉衣使臥底
  • 說明她的陣營選擇背後的個人動機與最終結局
