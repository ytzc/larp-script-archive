# 賈三娘 GM Notes

> **GM Only — 本文件包含賈三娘所有秘密、真實身份、任務內容及GM知道但玩家不知道的全部資訊。**

## GM Summary

賈三娘是綉衣使的在地情報樞紐，表面是客棧老板娘、實際是密探「歸雪」。她是綉衣使二號殺手，陣營固定、立場忠誠，無法倒戈。她的主要戲劇張力來自：她與殘手（嚴逸）之間存在多年的把柄關係、她曾親手重傷冷降塵之父冷佛林、她在凶案當晚親自移動了冷降塵的屍體。**此前GM文件中關於「感情線可能動搖立場」的描述來自OCR錯誤，原文並無此設定，請勿採用。**

## Public Identity

- **角色名：** 賈三娘
- **對外身份：** 風韻猶存的客棧老板娘，34歲，身着一袭紅衣
- **對外說法：** 原本在盤興口經營成衣坊，後投資客棧，生意清淡，賠了不少錢
- **客棧功能：** 她的客棧是所有人聚集的場所，天然佔有情報優勢

## Hidden Background（GM Truth）

1. **出身：** 自幼流浪街頭的小乞丐，8歲時被起朝權臣鄭木相中，帶入綉衣使秘密訓練營。真實姓名不詳，「歸雪」亦只是綉衣使代號。
2. **訓練：** 在訓練營10年，習得暗殺、暗器、制毒、蠱惑等術。14歲開始外出執行任務，升遷迅速，21歲鄭木建立起朝後成為綉衣使二號殺手。
3. **殺何彦一家（大夏元始三年，16歲）：** 她與殘手共同刺殺濟陽縣令何彦一家，任務完成後殘手私自放過了何彦的幼女。她以此作為把柄，但一直未動用，因尚不足以撼動殘手地位。
4. **擊殺冷佛林（龍瑞二年）：** 她是刺殺鹤須軍大統領冷佛林行動的參與者之一，親手以茶靈毒匕首刺入冷佛林右肋，致其中毒身亡。行動中被冷降塵以缚阳掌擊中左肩，留下永久黑紫色掌痕，此後慣穿深衣遮掩。
5. **混入玄甲軍：** 龍瑞四年宛平府一役後，奉命臥底玄甲軍，取得統帥旭敏芝信任，重新繪製了玄甲軍布防圖。旭敏芝還派她在盤興口開設天福客棧作為情報據點。
6. **無感情線：** 原GM文件中「感情線」條目為OCR錯誤所致，原文劇本中並無此設定。賈三娘的個人動機完全圍繞忠於鄭木與綉衣使。

## Role in Main Plot

1. **在地情報樞紐：** 掌握玄甲軍布防圖，情報網絡覆蓋盤興口
2. **接頭任務：** 等待殘手（嚴逸）前來接頭，共同執行任務
3. **凶案關鍵人物：** 凶案當晚親自移動冷降塵屍體至庫房（詳見凶案時間線）
4. **前朝追殺任務：** 收到線人密信，盤興口有大夏前朝重臣隱匿，須查明並除之

## Relationship to Murder Case

賈三娘是冷降塵屍體被移至庫房的直接執行者，但她本人並非凶手。

- **戌時（20:18）：** 她在大堂看到嚴氏與金四刀交談，起疑嚴氏夫婦，決定去冷降塵房間探聽。
- **戌時（20:25）：** 以飛虎爪攀爬天字四號房窗框，發現冷降塵不在，在房梁等待。後見冷降塵返房，與刁五儿打鬥，遂退回廚房等待。
- **亥時（21:00）：** 再返天字四號房窗邊，見刁五儿已離去，冷降塵昏迷倒地、衣服有血跡，房中有赤霞（綉衣使特製迷香）。判斷是同僚動手。
- **亥時（21:00–21:10）：** 扛起冷降塵，從二樓拖至庫房，開鎖入內。庫房內亦有赤霞，她吸入兩口後頭昏，倉皇撤退。撤退時發現冷降塵手指微動（可能仍有氣息）。
- **亥時（21:10）後：** 踉蹌返回自室，因赤霞效果昏睡，直至清晨余哥兒呼叫才起身。

**GM注意：** 她移動屍體的行為可能成為其他角色懷疑她的關鍵線索。她知道凶器是赤霞，知道是綉衣使同僚動手，但不知道具體是誰（雖然懷疑嚴氏夫婦）。

## Goals and Win Conditions

### 情景一任務

1. 盡可能對疑似敵對陣營的人隱瞞綉衣使真實身份
2. 盡可能拉攏人，鞏固朝廷勢力
3. 隱瞞暗殺濟陽縣令何彦一家及擊殺冷佛林的事實
4. 找到綉衣使內應並完成接頭
5. 盡可能獲得更多物品

- 問題1：上峰派誰來和你接頭？（客棧中的名字）

- 問題3：誰和舊朝大夏有直接或間接關係？至少說出2個名字

- 參與線索討論，盡可能投出真凶

2. 根據陣營牌正面任務行動

## What This Character Knows

- 她是歸雪，綉衣使密探（自知）
- 嚴逸是殘手，是她等待接頭的上司（需通過手勢確認）
- 嚴氏是嚴逸之妻（表面認知）；她懷疑嚴氏夫婦是綉衣使的同僚
- 冷降塵即「張猛」（她認出了他，因為綉衣使有各寇俠首領畫像，且她曾與冷降塵交手）
- 綉衣使手勢符號：分為內部成員手勢和線人手勢兩種；另有「三短一長」叩擊方式
- 庫房的赤霞迷香是綉衣使特製，說明有同僚動手
- 庫房鑰匙：她手中有一把，余哥兒有一把備用（備用鑰匙後被盜）
- 收到線人密信：盤興口郊有大夏前朝重臣偽裝農人隱居

## What Only GM Knows

- 賈三娘移動屍體是凶案的關鍵行動之一——她雖非凶手，但她的舉動使屍體出現在庫房
- 她左肩的掌痕是缚阳掌所留，這是冷降塵（或知情者）認出她真實身份的潛在線索
- 她16歲時與殘手同赴何彦刺殺任務，親眼見殘手放過幼女，這個把柄雙方都清楚
