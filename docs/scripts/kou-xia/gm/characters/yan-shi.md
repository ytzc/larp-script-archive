# 嚴氏 GM Notes

> **GM Only — 本文件包含嚴氏所有秘密、真實身份、任務內容及GM知道但玩家不知道的全部資訊。**

---

## GM Summary

嚴氏是本遊戲**複雜度最高、情感衝擊最強**的角色。她同時是：
- **凶手**（奉命殺死冷降塵）
- **悲劇受害者**（幼年失憶、被訓練為殺手的大夏皇族後裔）
- **最大彩蛋的核心**（金四刃之失散妹妹）

GM 必須熟悉她的所有面向，才能在復盤時完整呈現情感衝擊。

## How This Character May Affect the Game

1. **盤凶環節**：她是凶手，壓力最大，需要一邊參與討論一邊自保
2. **陣營搏殺**：線衣使方核心成員，戰力較弱（HP20）但陣營意義重大
3. **情感衝擊**：若在搏殺中面對金四刃，或身份被揭穿——觸發兄妹彩蛋
4. **復盤高潮**：兄妹真相是整場最大的情感爆點

---

## Hosting Notes

- 扮演嚴氏的玩家應有**能守口如瓶**、能承受壓力的性格（凶手角色難度高）
- 建議在發本前確認玩家理解凶手角色的壓力
- 搏殺環節中注意嚴氏與金四刃的位置——若發生直接對決，應觸發特殊演出

---

## Reveal / Debrief Notes

**復盤公告順序（嚴氏相關）：**
1. 宣布凶手：嚴氏（栾绛 / 何乃清）
2. 說明作案動機（嚴逸指使，滅口冷降塵）
3. 公布嚴氏真實身份（何乃清，大夏皇族後裔，失憶被訓練為殺手）
4. **壓軸彩蛋**：嚴氏 = 金四刃之失散妹妹

**注意**：建議在揭露兄妹真相前，先確認兩位玩家的心理狀態是否穩定，確保最大情感衝擊。
