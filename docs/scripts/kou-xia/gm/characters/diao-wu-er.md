# 刁五兒 GM 完整資料

> **GM ONLY — 請勿讓玩家看到本文件。**
> **已依 materials/kou-xia/source/diao-wu-er.md 全面核對確認（2026-06-22）。絕學卡：【卸器】（情景三）。真實身份：冷鈺（冷佛林之女），性別彩蛋為復盤最後揭曉。**

---

## GM Summary

刁五兒是本場遊戲中**情感核心最強烈的角色之一**。她並非任何陣營的正式成員，而是一個以女扮男裝、孤身入局的女兒，唯一的初始目的是為父（冷佛林）報仇——殺死她認為的兇手：義兄冷降塵。她的故事弧線從「復仇」走向「真相揭曉後的重新選擇」：與冷降塵的密室對峙中得知父親死亡另有真凶，並從義兄手中接過近衛軍兵符，讓她在陣營搏殺環節中的最終站隊充滿戲劇張力。

她還承載著一個**輕鬆的彩蛋**——她的真實性別（女性）在遊戲結尾才公開，是復盤時的趣味揭曉時刻，可以在氛圍緊張的結局後為全體玩家提供一個情緒出口。

## Public Identity（公開身份）

- **名稱**：刁五兒
- **外貌**：瘦小乾枯，黑布裹面的漢子，客商打扮（22歲）；全身皮膚潰爛，脖頸纏著厚厚繃帶，聲音嘶啞如男人
- **公開職業**：客商隨行護衛（與冷降塵〔張猛〕、金四刀同行採買鶴紅草）
- **公開性別**：男（偽裝）
- **表面個性**：沉默寡言，讓人不舒服的古怪感，其他人傾向迴避他

## Hidden Background（GM 真相）

- 真名冷鈺，原**鶴須軍**大統領冷佛林之女。
- 冷降塵（化名張猛）是她的**義兄**——本名丁典，原為敵將丁崧之子。丁崧因放走冷佛林入獄後，託人請冷佛林照顧其子。丁典在鶴須軍中成長後被冷佛林收為義子，改名冷降塵，冷鈺打心底把他當親哥哥。
- 龍瑞二年，冷佛林與冷降塵同行出發籌謀大事，途中遭人伏擊。冷降塵帶回冷佛林遺體，聲稱十數人圍攻，但本身毫髮無傷。冷鈺懷疑義兄謀殺父親、覬覦統領之位。
- 她趁機混入綉衣使，成為**綉衣使民間線人**，由一名戴面具的高手傳授她**無相功法**。此功法提高武功極快，但**副作用**是皮膚潰爛脫落，最終臉上留下可怖疤痕（這解釋了她厚厚繃帶、黑布蒙面的外貌）。
- 以男性身份「刁五兒」重返鶴須軍，成為冷降塵最信任的近侍，等待動手機會。

## Role in Main Plot

刁五兒在主線中扮演**雙重間諜 / 情感牽引角色**：

- 她是鶴須軍的小頭目，同時是綉衣使的民間線人，掌握雙方手勢符號。
- 她最初的任務是**殺死冷降塵**，但兇案環節中冷降塵主動找她攤牌，並告知父親另有真凶，從而改變了她的動機軸。
- 她的最終站隊（寇俠或綉衣使）可以影響陣營搏殺結果——劇本明文她**可選任一陣營**。

## Relationship to Murder Case

**冷降塵謀殺案（主案）：**

- 刁五兒**不是**兇手——她在 20:30–20:50 與冷降塵在天字四號房密談，20:50 將冷降塵送出房門後回到自己房間，徹夜未眠。

## What This Character Knows（遊戲開始時刁五兒知道的）

- 父親冷佛林已死，她確信是冷降塵所殺（這是她的初始認知，尚未被修正）。
- 冷降塵是她的義兄（原名丁典），她打心底把他當親哥哥——這也是這段仇恨的殘酷性所在。
- 她是綉衣使線人，也是鶴須軍小頭目，掌握雙方手勢。
- 她感受到客栈某人身上有無相功法氣息，推測師父就在這寥寥數人之中。
- 她知道布告欄落款「人口丁」很古怪（此人可能另有深意）。

## What This Character Learns（兇案環節密談後）

- 冷降塵**並非**殺父真凶——父親是被一名戴面具、用毒匕首的刺客所殺；冷降塵用縛陽掌擊中刺客**左肩**，刺客傷後逃走。
- 冷降塵懷疑真凶就住在客栈的「地字…」（話說到一半停下）。
- 她從冷降塵手中接過**近衛軍兵符**（鶴須軍半數軍權）。

## What This Character Does Not Know

- 父親的具體真凶是誰，以及與冷降塵謀殺案是否同出一脈。
- 各角色的真實陣營與秘密任務。
- 面具師父的真實身份。

---

## What Only GM Knows

- 刁五兒真實性別為女性（冷鈺）——其他玩家直到復盤才知道。
- 她是冷佛林之女、冷降塵義妹這一點，在遊戲中其他玩家可能完全不知道。
- 她的身份曝光時機由GM控制，或由玩家自行選擇何時揭曉。
- 傳授她無相功法的面具師父，其身份是本遊戲的另一條謎線。

---

## How This Character May Affect the Game

1. **情感張力**：她本來要殺冷降塵，結果冷降塵先找她攤牌、交出軍權就被人殺了——這個反轉對玩家而言情感衝擊極大，GM可以關注玩家的反應。
2. **陣營天平**：她初始可選任一陣營，最終站隊可影響陣營搏殺結果，GM應注意兩陣營都可能積極拉攏她。
3. **線索提供者**：她知道冷降塵密談中提到的「刺客縛陽掌擊中左肩」細節，這是查明父親真凶的關鍵線索。
4. **近衛軍兵符**：她持有兵符，這是鶴須軍半數軍權的象徵，可能成為陣營談判的籌碼。
5. **性別彩蛋**：她的秘密不影響主線勝負，但是復盤時的重要趣味時刻。GM需要在正確時機揭曉（見 Reveal Notes）。

## Hosting Notes

- 刁五兒的玩家需要全程保持「男裝」表演，建議GM在遊戲前和玩家確認其意願及舒適度。
- 她的沉默個性（「古怪沉默讓人迴避」）是有意識的保護機制，玩家應理解這背後有深刻情感。
- 她的武功反噬（皮膚潰爛）是劇本設定，建議引導玩家在描述外貌時強調這一點，增加代入感。
- 她持有近衛軍兵符但不一定知道該如何使用，GM可適時引導她與其他鶴須軍相關角色互動。
- 她可選任一陣營——玩家可能在遊戲中期感到迷茫。GM可在適當時機給予輕微引導（「你的目標是真相和軍權，陣營只是手段」）。

## Reveal / Debrief Notes

**揭曉順序建議：**

1. 主線真相揭曉（冷降塵謀殺案凶手與主謀）
2. 父親冷佛林真凶揭曉（與冷降塵謀殺案的關聯）
3. **刁五兒真實性別「冷鈺」**——作為輕鬆彩蛋，在最後揭曉

**GM說法建議：**
「最後，我們的刁五兒，這個看起來沉默怪異的男護衛……其實是一位女俠！她的真名是冷鈺，原鶴須軍大統領冷佛林的女兒，女扮男裝前來為父報仇。大家鼓掌！」

**設計意圖：**
此彩蛋屬於「輕鬆」揭示，在重量級的主線真相之後提供情緒出口。不建議提前劇透，也不建議在緊張對決時機揭示，以免破壞氣氛。
