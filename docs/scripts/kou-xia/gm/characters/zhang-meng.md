# 張猛（NPC）— GM 專屬資料

> **重要**：張猛為 NPC，由 GM 或助理扮演，**無玩家版本**。本文件僅供 GM 使用。

## GM Summary（GM 扮演指引）

張猛是本場遊戲的主要 NPC，由 GM 或指定助理扮演。根據劇本，扮演者以**化名「孟猛」**出場（真實身份為**冷降尘**）。外觀特徵：魁梧黑須，豪爽無比的大漢，鏢客商人打扮，27歲。

GM 扮演此角色時需注意：
- 張猛的真實身份是冷降尘，但在本次任務中以「孟猛」的化名行動。
- 他的公開形象是鏢客／商人，不暴露復國組織身份。
- 在「凶案環節」，張猛扮演**屍體**：需攜帶3張「屍體線索」卡及一封書信，分別藏於身上不同位置，讓玩家搜查發現。完成線索擺放後，NPC 任務即結束，退場。

## Background（背景故事）

冷降尘原名丁勇，父親為大夏王朝重臣及大將（援預鈞林名將）。大夏末年，邓木把持朝政，父親因私庇寇侠首領冷俠林而遭告發，丁氏家族被迫流放、離散。冷降尘年幼時被送往冷俠林處寄養，從此改名冷降尘，在冷俠林的鶴駕軍中成長。

他自幼在鶴駕軍中打磨武藝，擅長父親傳下的「搏陽錘」功法。冷俠林待他如子，冷俠林之女冷姐與他情同兄妹。
