## GM Summary（GM 扮演指引）

張猛是本場遊戲的主要 NPC，由 GM 或指定助理扮演。扮演者以**化名「張猛」**出場（真實身份為**冷降塵**，幼名**丁典**）。外觀特徵：客商打扮，27歲。

GM 扮演此角色時需注意：
- 張猛的真實身份是冷降塵（鶴須軍大統領），但在本次任務中以「張猛」的化名行動。
- 公開身份：客商，和金四刀、刁五兒一同到天福客栈「採購鶴紅草」。
- **核心任務：帶節奏、炒場、增強現場活躍度**。
- 在「凶案環節」，張猛扮演**屍體**：攜帶3張「屍體線索」卡及一封書信，分別藏於身上不同位置，讓玩家搜查發現。完成線索搜證後，NPC 任務結束，退場。

## Background（背景故事）

冷降塵幼名丁典，親生父親為大夏王朝重臣及名將丁崧。大夏末年，權臣鄭木（郑木）把持朝政，民間興起了許多反抗鄭木的武裝勢力，被稱為「寇侠」。

丁崧抓住了寇侠鶴須軍首領冷佛林，卻發現對方義薄雲天、心懷大義，二人多次交談成為好友，最終丁崧決定**私放冷佛林**。此事被丁崧手下告發，鄭木得知後震怒，下令流放丁氏全家，派綉衣使押送。丁崧（農叟）拼盡全力將丁典（冷降塵）救出，並把他托付給冷佛林。父子就此別離。

冷佛林收丁典為義子，改名冷降塵，在鶴須軍中成長。冷佛林獨女冷鈺（刁五兒）小冷降塵五歲，二人情同親兄妹。

**冷佛林之死**：冷降塵21歲時，冷佛林秘密聯繫天下寇侠首領籌劃大事，途中遭伏擊。一名蒙面刺客用帶毒匕首刺傷冷佛林右肋，冷降塵一掌（缚阳掌）打在刺客**左肩**上，刺客落荒而逃，冷佛林毒發身亡。

**宛平大戰**：繼承鶴須軍大統領後，冷降塵與天下寇侠軍聯合攻打宛平府，然而與玄甲軍起了嫌隙，合作不默契，大敗。玄甲軍統帥旭晨在此役中戰死，冷降塵遠遠看見一個穿鶴須軍軍服的男子割下旭晨首級後趁亂逃走，知道是有人渾水摸魚，鶴須軍軍紀嚴明，絕不會行此卑鄙之事。

**金四刀（金四刀）**：重建鶴須軍時，冷降塵廣納人才，結識了表現出色的年輕將領金四刀，知道其真實身份是前朝大夏皇族旁支宗親，真名何乃武。給了他鶴須軍將領才有的碧綠玉佩（刻有羽毛痕跡），軍中只有二人各持一塊。

**刁五兒**：冷降塵為自身安全找了武功高強的近衛—刁五兒（身材矮小，黑布蒙面，身形佝僂，終日陰鬱）。**冷降塵收到密信後，知道刁五兒其實是失蹤多年的義妹冷鈺**，遊戲中將在第一輪私聊快結束時傳遞近衛軍兵符給她。

## 兵符傳遞台詞（劇本原文）

### 將虎贲軍兵符交予金四刀：
> 「世子若想復興大夏，單靠我鶴須軍的力量是遠遠不夠的。這兩塊兵符，一個是鶴須虎贲衛兵符，一個是近衛軍兵符。如今的鶴須軍聲勢大漲，都是世子的功勞，冷某是不敢搶功的。更何況世子乃大夏正統，冷某更是會全力支持世子的大計。只是鶴須軍原本就是冷某代義父領導，如今我已經找到了他的後人，這塊近衛軍兵符，自然要交到他後人的手上。至於此人是否能像冷某一樣支持世子，就要看世子的造化了。」

### 將近衛軍兵符交予刁五兒：
> 「這是我鶴須軍的近衛軍兵符，今日我將鶴須軍的半數軍權交給你。」

## 私聊話術指引（各角色）

| 角色 | 話術要點 |
|------|---------|
| 刁五兒 | 告知：「我收到密信，知道了你是我的義妹冷鈺。我想告訴你義父絕不是我殺的。義父當年被一個蒙面刺客刺傷右肋，由於匕首帶毒，義父毒發身亡。凶手逃走時左肩中了我的缚阳掌。」**不要告訴其他玩家刁五兒身份。** |
| 王思涵 | 若詢問旭晨之死：「我知道你是為何而來，不過我並非殺害旭晨的真兇，我鶴須軍中也不會有人幹出如此卑鄙的事。當年之事是有人借機挑撥，陷害於鶴須軍。」 |
| 農叟 | 「我對你有種莫名的親切感。」若農叟提到小木劍可以父子相認。若農叟沒有透露兒子名字丁典、也沒有提到小木劍，可以不作引導。相認之後可暗示「當年的事情是被他的部下陷害。」 |
| 嚴氏/嚴逸 | 聊江湖消息：「江湖上有傳聞，當年前朝皇族何氏，一夜之間被滅門。當時何氏夫婦還有一大家子的尸體都被發現了，現場唯獨少了他們家兒子和女兒的尸體，不知道是不是另有造化。」 |
| 賈三娘 | 若提起殺冷佛林之事：「我覺得你長得很眼熟。」其他玩家問起賈三娘：「我看這位老板娘長得十分眼熟，也許是我曾經見過的人。」 |
| 王順 | 「這個人頗有城府，不可小覷。」（劇情上已了解其所作所為，但無確凿證據） |
| 金四刀 | 可提及：「我知道你的身份，是大夏後裔一族，何氏的血脈。」對其他玩家不說出此內容。 |

**不需要隱藏、可透露給所有人的線索：**
- 「我在盤興口發現了前朝大夏將領的蹤跡，應該就在我們之中。」
- 「這些日子天福客棧會有綉衣使來接頭。」
- 「我知道現在客棧中應該有玄甲軍的實際統帥，你知道是誰嗎？」
- 「何氏滅門慘案我有所耳聞，據說當時何氏夫婦還有一大家子的尸體都被發現了，現場唯獨少了他們家兒子和女兒的尸體。」
- 可提及丁崧往事，用「聽說過嗎，前朝大夏有個將領丁崧是被部下陷害的」話術。

### 書信原文

> 降塵兄，中元節將至，可否請君至玄甲軍中，共祭宛平大戰數萬兄弟，且當前朝廷昏庸，當盡快定下再舉之策。此乃密謀，我等將化身前往盤興口天福客棧等待降塵兄。

**GM注：** 書信落款為玄甲軍大統領（旭敏芝，即王思涵），是引冷降塵赴約的「鴻門宴」邀請函。玩家在搜查屍體時可發現此信，作為凶案線索之一。

## Hosting Notes

- NPC 全程以「張猛」化名，對外不暴露鶴須軍大統領身份
- **一定不要暴露金四刀是大夏何氏皇族後裔**
- 手勢符號可畫到便簽上使用，也可在對方掌心直接作畫，遊戲全程可用
- 與農叟相認時機視玩家互動而定，不強制推動；農叟提到小木劍則可相認
- 江湖密行環節：作為普通玩家參與，獲勝正常抽取獎勵
