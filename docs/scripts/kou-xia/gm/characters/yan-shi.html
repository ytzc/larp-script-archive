主持人提示

角色分配建議：扮演嚴氏的玩家應有能守口如瓶、能承受壓力的性格（凶手角色難度高）。建議在發本前確認玩家理解凶手角色的壓力。
  • 搏殺環節中注意嚴氏與金四刃的位置——若發生直接對決，應觸發特殊演出
  • 她是凶手，盤凶環節壓力最大，需要一邊參與討論一邊自保
  • 線衣使方核心成員，戰力較弱（HP20）但陣營意義重大

復盤公告順序

  1. 宣布凶手：嚴氏（栾绛 / 何乃清）
  2. 說明作案動機（嚴逸指使，滅口冷降塵）
  3. 公布嚴氏真實身份（何乃清，大夏皇族後裔，失憶被訓練為殺手）
  4. 壓軸彩蛋：嚴氏 = 金四刃之失散妹妹
揭露兄妹真相前，先確認兩位玩家的心理狀態是否穩定，確保最大情感衝擊。
