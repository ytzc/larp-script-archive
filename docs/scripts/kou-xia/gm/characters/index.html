復盤公告順序（GM 參考）

  1. 宣布凶手：嚴氏（欒絳 / 何乃清），說明動機（嚴逸指使，亥時 21:10 茶靈毒匕首插入背心）
  2. 公布嚴氏真實身份（何乃清、大夏皇族、失憶被訓練、臉上疤痕是嚴逸掌風而非胎記）
  3. 公布各角色真實身份（由簡到複雜）
  4. 壓軸彩蛋：嚴氏與金四刀是失散兄妹（確認兩位玩家狀態後揭露）
  5. 輕鬆彩蛋：刁五兒真實性別是女性（冷鈺）
  6. 開放提問與分享感受

GM 核心關係速查

關係 細節（GM 視角）
嚴氏 ↔ 金四刀親妹妹 ↔ 親哥哥（雙方均不知）。金四刀在廚房認出嚴氏脖頸青色胎記。
嚴氏 ↔ 嚴逸嚴逸殺了她全家，帶走她訓練成殺手，臉上紅疤是嚴逸掌風所致（謊稱胎記）。
農叟 ↔ 嚴逸（右肩）農叟（丁崧）在何家當夜從後打中嚴逸右肩，留下永久紫黑掌印。
農叟 ↔ 冷降塵農叟的親生兒子丁典 = 冷佛林義子冷降塵（NPC 張猛）。情景二父子相認。
農叟 ↔ 王順王順（龐萬金）是農叟舊部叛徒，告發農叟導致流放。認出農叟後設赤霞香庫房圈套試圖滅口。
刁五兒 ↔ 冷降塵義兄妹（冷佛林義子 + 親生女）。冷降塵知道刁五兒是冷鈺，在私聊中相認。
刁五兒 ↔ 賈三娘賈三娘（歸雪）5年前用茶靈毒匕首刺殺了刁五兒的父親冷佛林。
刁五兒 ↔ 嚴逸嚴逸奉命訓練了刁五兒，給了她半部刪去調和內容的《無相功法》，造成嚴重反噬。刁五兒能感受到嚴逸的無相功法氣息。
王思涵 ↔ 王順王思涵的兄長旭晨在宛平府大戰中被王順（偽裝成鶴須軍兵丁）用左手短柄手戟殺害割頭。
