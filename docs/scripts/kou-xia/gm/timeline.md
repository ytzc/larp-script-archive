## 背景世界觀

故事發生在架空古代，當前朝代為「大順」，前朝為「大夏」。

- **大夏**：前朝，已被大順所滅。大夏皇族後裔仍有人潛伏民間，伺機復國。
- **大順**：現在的朝廷，設有「綉衣使」作為秘密特務組織，負責追查大夏殘黨與江湖異人。
- **寇俠組織**：以「寇俠」為名，實為大夏皇族後裔及其盟友組成的復國地下組織。

## 核心人物關係（GM視角）

```
金四刀（何乃武）─── 哥哥 ───→ 嚴氏（何乃清）
                                   ↑
                              何乃清已失憶，
                            不記得自身為大夏皇族
```

- **嚴逸**（綉衣使三號「殘手」）負責追殺大夏殘黨，派遣**嚴氏**（何乃清 / 栾绛）執行殺人任務。
- 嚴氏原為大夏皇族，後因意外失憶，被綉衣使洗腦訓練為殺手「栾绛」，並以「嚴氏」身份臥底。
- 金四刀（何乃武）是嚴氏失憶前的親哥哥，二人身份互相不知。

## 各陣營目標（GM視角）

### 寇俠陣營
- 領袖：金四刀（何乃武）
- 核心成員：視劇本選邊
- **勝利條件**：陣營搏殺後寇俠方存活人數多於綉衣使方，且金四刀存活

### 綉衣使陣營
- 領袖：嚴逸（殘手，三號）
- 核心成員：嚴氏（栾绛）、賈三娘（歸雪）
- **勝利條件**：陣營搏殺後綉衣使方存活人數多於寇俠方，且嚴逸存活

## GM掌握的關鍵信息摘要

1. **真正的凶手是嚴氏**，但她是奉命行事（嚴逸指使）
2. **嚴氏與金四刀是失散兄妹**，雙方均不知情（最大隱藏彩蛋）
3. **刁五兒**實際上是女性（表面是男性打扮的冷鈺）
4. **農叟（丁崧）** 與 **王順（龐萬金）** 是舊主舊部關係，有歷史情感連結
5. **賈三娘（歸雪）** 的真實任務是輔助嚴逸，但感情線可能影響立場
