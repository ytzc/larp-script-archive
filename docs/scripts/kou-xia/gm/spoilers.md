## 凶案真相

### 凶手
**嚴氏（真實身份：栾绛 / 何乃清）**

### 被害人
**冷降塵**

### 作案動機
嚴氏奉綉衣使三號「殘手」（嚴逸）之命，消除知情者冷降塵，防止其洩露綉衣使在此地的行動計劃。

### 凶器
詳見線索卡（盒裝物品）。

### 隱藏真相（最大彩蛋）
- 嚴氏（何乃清）是金四刃（何乃武）多年前失散的親妹妹。
- 兩人均不知對方身份。
- 若金四刃在陣營搏殺中與嚴氏對決，或嚴氏被揭穿身份，應觸發感情結局。

## 各角色個人真相

### 金四刃（何乃武）
- 大夏最後一代皇族旁系後裔
- 以「金四刃」武者身份隱藏，實為寇俠組織在此次聚會的聯絡人
- 尋找多年失散的妹妹何乃清（即嚴氏）
- **最大衝擊點**：在復盤時得知嚴氏即為何乃清

### 嚴逸（殘手）
- 大順綉衣使三號，此行負責清除大夏殘黨
- 指使嚴氏殺死冷降塵
- 目標：確認並消滅金四刃（何乃武）

### 賈三娘（歸雪）
- 綉衣使密探，表面是普通女子
- 輔助嚴逸收集情報
- 個人感情線可能使立場動搖（見個人劇本）

### 農叟（丁崧）
- 大夏舊朝武將，帶領殘部歸隱多年
- 無初始陣營，被王順（龐萬金）發現身份後須重新抉擇
- 個人目標：守護舊部、不讓大夏遺民再受迫害

### 王思涵（旭敏芝 / 阳敏芝）
- 寇俠玄甲軍統師，武力超群
- 對大夏復國半信半疑，主要在乎舊部族人安危
- 可加入任一陣營，視私聊結果而定

### 嚴氏（栾绛 / 何乃清）
- **全場最大秘密人物**
- 大夏皇族後裔，幼年失憶後被綉衣使訓練為殺手
- 以「嚴氏」身份接近目標，實際執行此次殺人任務
- 與金四刃（哥哥）互不相識
- **凶手**，但從某種角度是悲劇受害者

### 王順（龐萬金）
- 前朝武將丁崧（農叟）的舊部
- 認出農叟後，個人目標轉為保護舊主
- 無初始陣營，忠誠度是最大變數

### 刁五兒（冷鈺）
- 冷佛林之女，**實際上是女性**（全程以男性裝扮出現）
- 前來為父（冷佛林，已故）追查真相
- 冷降塵是其父的舊識，死亡與其個人目標相關
- 無初始陣營，可選邊

## 結局說明

### 大結局 A：寇俠陣營勝

**條件：** 陣營搏殺後寇俠方存活人數 ≥ 綉衣使方，且金四刃（何乃武）存活

**結局：**
> 寇俠組織成功守住核心人物，大夏復國計畫獲得喘息空間。金四刃帶領存活的盟友，化整為零繼續潛伏……

各角色個人結局（寇俠勝）：

| 角色 | 個人結局摘要 |
|------|------------|
| 金四刃 | 寇俠組織暫時安全；若得知嚴氏身份，則留下心結 |
| 嚴氏 | 若被揭穿且存活，可能選擇回歸大夏（兄妹重逢線） |
| 農叟 | 選擇寇俠方者：重歸故主麾下，守護餘生 |
| 王思涵 | 繼續以玄甲軍統師身份協助組織 |
| 刁五兒 | 為父報仇，冷降塵真相大白，可功成身退 |
| 王順 | 隨農叟（舊主）行動，守護義氣 |
| 賈三娘 | 若倒戈寇俠：以全新身份重生；若堅守綉衣使：戰敗被俘或撤退 |
| 嚴逸 | 失敗，撤退或被擒；綉衣使暫失此局 |

---

### 大結局 B：綉衣使陣營勝

**條件：** 陣營搏殺後綉衣使方存活人數 ≥ 寇俠方，且嚴逸（殘手）存活

**結局：**
> 綉衣使成功清除大夏殘黨核心，大順朝廷的威脅暫時解除。嚴逸帶著任務完成的回報，踏上歸程……

各角色個人結局（綉衣使勝）：

| 角色 | 個人結局摘要 |
|------|------------|
| 嚴逸 | 任務完成，升遷；若得知嚴氏身份，則留下內心糾葛 |
| 嚴氏 | 功成，但若恢復記憶：深陷矛盾，殺了哥哥或選擇自裁 |
| 賈三娘 | 綉衣使獲勝，維持臥底身份繼續任務 |
| 金四刃 | 戰敗，若存活則繼續流亡；若嚴氏揭曉身份，兄妹陰陽相隔 |
| 農叟 | 舊朝最後一段武將情懷就此落幕 |
| 王思涵 | 見風使舵成功者功成身退；守義者戰敗 |
| 刁五兒 | 父仇得報（真相揭曉），但大夏覆滅 |
| 王順 | 隨農叟（舊主）同命 |

## 復盤公告順序建議

1. 宣布凶手：**嚴氏（栾绛 / 何乃清）**，說明動機
2. 公布各角色真實身份（由簡到複雜）
3. 最後揭露：**嚴氏與金四刃是失散兄妹**（高潮彩蛋）
4. 公布刁五兒真實性別（輕鬆彩蛋）
5. 開放提問與分享感受

> **主持提示：** 兄妹真相建議作為壓軸。先確認兩位玩家均已冷靜，讓彩蛋有最大情感衝擊。
