劇本整理備註 — OCR 品質整體說明

本頁彙整所有角色劇本的 OCR 提取結果與整理備註，僅供 GM 參考。所有內容來自 2026-06-16 整理的 .md 稿件，以及 CHECK_REPORT.md 品質審查報告。

標記說明：

  • [需要人工確認] — OCR 信心度低，建議對照實體書本確認。
  • [疑似缺漏] — 段落明顯不完整，應有後續文字未被讀取。
  • [需要補拍] — 整節無法讀取，需重新拍攝該頁才能取得內容。

重要說明： 整理稿保留原文語言（簡體不轉繁體，繁體不轉簡體）；OCR 誤字保留原樣並標注，不擅自更正；不添加原文中未有的劇情細節。GM 隔頁說明（「本段GM閱讀，太子殿下，請回避！」）及GM指令頁均不包含在玩家整理稿中。

跨角色 GM 真相彙整

以下為本場最核心的 GM 層真相，玩家劇本中各自掌握片段，GM 需全盤掌握。

冷降塵之死：凶手與主謀

直接凶手： 嚴氏（栾绛）

幕後主謀： 嚴逸（殘手）

嚴逸劇本頁3紅色加粗明確寫道：「你指使了『嚴氏（栾绛）』殺死了冷降塵——因為冷降塵知道太多，他必須消失。」

GM注意： 嚴氏頁6/8的凶手行動指令段落因裝訂陰影遮蔽，需補拍確認具體道具、時機與方式。

嚴氏的真實身份：何乃清

嚴氏（栾绛）真名是何乃清，大夏皇族後裔，金四刀（何乃武）失散多年的妹妹。

  • 嚴氏身上有青色胎記（頸後），與金四刀描述的「頸後青色胎記」完全吻合。
  • 嚴氏幼年記憶缺失——被嚴逸訓練為殺手前的記憶空白，不知道自己真實身份。
  • 嚴逸當年奉命殺害何彦全家，但對女童（何乃清）刀下留情，謊報全數伏誅。他三年前認出了長大的何乃清，以「妻子」身份收用，訓練為殺手「栾绛」。
  • 嚴逸明確不知道嚴氏（何乃清）與金四刀的兄妹關係（嚴逸頁4）。

冷佛林／冷俠林：同一人？

刁五兒劇本稱父親為「冷佛林」，張猛劇本稱寇俠首領為「冷俠林」。兩個名字疑似同一人，OCR誤讀所致。

冷降塵（張猛）是冷俠林收養的義子；刁五兒（冷鈺）是冷俠林親女——二人如義兄妹。

待確認： 「冷佛林」與「冷俠林」是否確為同一人，需對照原本核對。

農叟（丁崧）與王順（龐萬金）的關係

農叟真名丁崧，大夏托孤重臣，目前無陣營，以農夫身份隱居。

王順真名龐萬金，曾是丁崧部下小將，但年輕時曾告發丁崧給綉衣使（希望藉此晉升，未遂）。後再次遇到農叟，陷入忠義與綉衣使任務的兩難。

王順以「秀明」（農叟之女）作為女兒身份掩護混入集會，秀明本人不知真相。

金四刀（何乃武）的身份與任務

金四刀（何乃武），26歲，大夏皇族九世孫，寇俠聯絡人。通過王思涵（旭敏芝）加入寇俠。

多年來尋找失散妹妹何乃清（頸後青色胎記），本次聚會後計劃請王思涵借用情報網打探消息。

金四刀視嚴氏「眼神令他隱隱不安，彷彿在哪裡見過」，但未識破兄妹關係——此為本場最大伏筆之一。

王思涵（旭敏芝）：玄甲軍統師

王思涵真名旭敏芝（或阳敏芝，簡繁差異待確認），21歲，歌姬身份掩護，實為寇俠玄甲軍統師。

金四刀通過她加入寇俠，二人互信，金四刀確認「知道王思涵是寇俠玄甲軍統帥」。

GM注意： 王思涵劇本頁2–10 OCR品質最差（幾乎全為佔位符），是本批整理稿中內容最匱乏的角色，必須優先補拍。

賈三娘（歸雪）：感情線待確認

賈三娘真名歸雪，34歲，客棧老板娘身份掩護，綉衣使密探，協助嚴逸（殘手）行動。

賈三娘有一條感情線，某人讓她的心動搖，可能影響對綉衣使的忠誠。感情線對象在頁7完全無法辨讀（[需要補拍]）。依GM關係矩陣推測可能為王思涵，但尚未確認。

嚴逸（殘手）的核心秘密

嚴逸是綉衣使頂級殺手「殘手」，書生身份掩護，技能「無相功法」（武力+3，反噬每回合-2HP）。

嚴逸的四大秘密（頁7）：

  1. 他是「殘手」——一旦被揭穿，任務失敗。
  2. 他是冷降塵命案的幕後主謀，指使嚴氏動手。
  3. 他當年沒有殺死那個女童（即何乃清），但謊報全數伏誅——連上司都不知道。
  4. 他對嚴氏的感情是否已超出「工具」的範疇，是他最不想面對的問題。

張猛（NPC）：冷降塵的前史

張猛真名冷降塵，化名孟猛，幼名丁勇，27歲。親父為大夏重臣，被家族流放後由寇俠首領冷俠林收養，改名冷降塵。

NPC職責：扮演死者，持有3張屍體線索（C1–C3）及1封書信（L1），藏身上不同位置，凶案環節後退場。

GM注意： 頁4–6 GM指引（紅色文字）、人物關係密碼牌、玩家互動分配說明全部因印刷過淡而遮蔽，需對照DM手冊。

五、OCR提取格式說明

所有 .md 整理稿中使用以下標記：

  • [需要人工確認] — 此處OCR信心度低，原文詞語可能被誤讀，建議對照實體書本確認。
  • [疑似缺漏] — 此處段落明顯不完整（句子截斷、內容突然中斷），應有後續文字未被讀取。
  • [需要補拍] — 此節整個無法讀取（大範圍裝訂陰影或印刷過淡），需重新拍攝該頁才能取得內容。

重要說明：

  • 整理稿保留原文語言（簡體不轉繁體，繁體不轉簡體）
  • OCR誤字保留原樣並標注，不擅自更正
  • 不添加任何原文中未有的劇情細節
  • GM隔頁說明（「本段GM閱讀，太子殿下，請回避！」）及GM指令頁不包含在玩家整理稿中
