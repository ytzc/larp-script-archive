  3. 頁003–006 OCR品質：中等，內側陰影造成部分遺失，但核心目標段落基本完整。

王順 整理備註

  1. 角色核心：王順（龐萬金），43歲，大夏重臣丁崧部下小將。曾告發丁崧給綉衣使，以為能晉升，結果未得到預期地位，成為底層線人。後再次遇到農叟（丁崧），陷入忠義與綉衣使任務的兩難。以秀明為偽女，混入本次集會。
  2. 頁2（你的故事）OCR品質：中等，多處誤讀，但核心劇情（告發丁崧、未得晉升、成為線人）可辨。
  3. 頁3–9 OCR品質：中等偏差，大量行被裝訂陰影截斷，以片段形式保留。
  4. 秀明關係：農叟的女兒，被王順帶著當掩護女兒，秀明本人不知真相。
  5. 核心抉擇：綉衣使任務（監視冷降塵/寇俠）vs 保護舊主農叟——這是本角色的主要張力。
  6. 陣營：綉衣使（線人），但因舊主農叟的出現而搖擺。
  7. 冷降塵之死：王順不知凶手，只知與「寇俠內部權力爭奪」有關（此為其視角，非GM真相）。

五、OCR提取格式說明

所有 .md 整理稿中使用以下標記：

  • [需要人工確認] — 此處OCR信心度低，原文詞語可能被誤讀，建議對照實體書本確認。
  • [疑似缺漏] — 此處段落明顯不完整（句子截斷、內容突然中斷），應有後續文字未被讀取。
  • [需要補拍] — 此節整個無法讀取（大範圍裝訂陰影或印刷過淡），需重新拍攝該頁才能取得內容。

重要說明：

  • 整理稿保留原文語言（簡體不轉繁體，繁體不轉簡體）
  • OCR誤字保留原樣並標注，不擅自更正
  • 不添加任何原文中未有的劇情細節
  • GM隔頁說明（「本段GM閱讀，太子殿下，請回避！」）及GM指令頁不包含在玩家整理稿中
