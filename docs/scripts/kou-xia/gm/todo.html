嚴逸是否知道嚴氏的大夏皇族血統？
已從 source/jia-san-niang.md 確認（2026-06-22）
已完成
結論嚴逸（殘手）知道嚴氏是何乃清（大夏皇族）。
來源賈三娘劇本：大夏元始三年，殘手（嚴逸）與賈三娘（歸雪）一同執行暗殺何彥全家的任務。任務完成後，殘手一反常態，主動選擇放走何乃清（小女童），對賈三娘說「放過她吧，算我欠你一個人情」。
深層含義嚴逸對何乃清有複雜的個人情感，特意保住她性命。後何乃清被繡衣使收容訓練成「嚴氏/欒絳」，嚴逸是否告知她的真實身份則不確定（需看嚴逸劇本）。

王思涵 ↔ 農叟 / 王順 的直接歷史關係
已從 source mds 確認（2026-06-22）
已完成
王思涵↔農叟無直接歷史關係。丁崧（農叟）是大夏朝廷將領，旭氏（王思涵家族）是地方鄉紳組成的寇俠玄甲軍，二人不是同袍，過去未曾相識。
王順↔農叟王順（龐萬金）是丁崧（農叟）舊部小校，後背叛告發丁崧。農叟在情景二確認王順就是龐萬金（「他不和自己相認？」）。這是農叟的核心仇恨對象。
王順↔王思涵王順（龐萬金）是玄甲軍臥底，直接上司是旭敏芝（王思涵）。他殺了她的兄長旭晨（宛平府大戰中扮成鶴須軍兵丁，用左手揮短柄手戟），是她的殺兄仇人。
