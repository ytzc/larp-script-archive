📷 補拍清單

各角色需要重新拍攝的頁面、原因與優先順序

補拍後請更新對應的 md 整理稿，再重新生成 HTML。 OCR 提示詞模板請見 OCR 提示詞指引。

🔴 優先級1（遊戲執行關鍵）

這些頁面缺失會直接影響遊戲可執行性，開本前必須補拍。

賈三娘 — 頁2–9（幾乎全部）

賈三娘 頁2–9 🔴 高
缺失：整個角色劇本的核心內容——人物背景、任務、人際關係、感情線、勝利條件、特殊行動等，幾乎全不可讀。
原因：裝訂陰影嚴重，頁面內側整列文字遮蔽。感情線對象姓名（角色核心）完全無法辨讀。

王思涵 — 頁2–10（幾乎全部）

王思涵 頁2–10 🔴 高
缺失：除封面外，整個劇本幾乎全不可讀。技能、勝利條件、陣營選擇、任務、人際關係全部缺失。
原因：OCR提取時文字幾乎全被標記為 [OCR?]，推測為印刷品質差（字體過小/過淡）＋裝訂陰影雙重問題。補拍時建議以更高解析度（2x scale 以上）重拍。

嚴氏 — 頁6、頁8（凶手行動指令）

嚴氏 頁6（下半）、頁8（上半） 🔴 高
缺失：嚴氏殺死冷降塵的具體指令——道具使用方式、時機、方法。這是本案凶手還原的核心。
原因：裝訂陰影遮蔽頁面右欄內側。嚴逸劇本已明確寫出「嚴逸指使嚴氏謀殺冷降塵」，但嚴氏自身劇本中的行動細節缺失。

農叟 — 頁7（農叟特殊規則方框）

農叟 頁7（技能方框） 🔴 高
缺失：【農叟之力】完整技能說明，以及農叟在搏殺環節的特殊規則。
原因：農叟 HP 40 / 武力 6 為全場最高，特殊規則直接影響搏殺環節平衡，缺失可能導致規則爭議。

🟡 優先級2（劇情理解重要）

缺失會造成劇情邏輯不完整，建議開本前確認。

刁五兒 — 頁7（技能數值）

刁五兒 頁7（技能方框） 🟡 中
缺失：HP 和武力確切數值（GM 備注估計 HP≈25，武力 3–6，仍需原本核對）。
原因：裝訂陰影遮蔽技能方框。搏殺環節需要準確數值。

農叟 — 頁001（開場故事）

農叟 頁001（故事段） 🟡 中
缺失：「汲洲任務」開場故事段落，OCR誤讀嚴重，現有文字多處語義不通。
原因：農叟/何乃青的關係是跨角色重要線索，需確認「汲洲任務」的具體指令及「何乃青」是否即金四刃。

王順 — 頁3–9（關係與抉擇段落）

王順 頁3–9 🟡 中
缺失：人際關係條目（與農叟的重逢細節）、綉衣使任務具體說明、核心抉擇（忠義 vs 任務）相關段落。
原因：王順的核心張力是「舊主農叟 vs 綉衣使任務」，多行關鍵文字被裝訂陰影截斷。

嚴逸 — 頁8（秘密問題3/4）

嚴逸 頁8（秘密問題下半） 🟡 中
缺失：秘密問題3「栾绛真實姓名及是否有親人」和問題4「綉衣使線人號碼」的 GM 標準答案。
原因：嚴逸是凶案關鍵——他指使嚴氏謀殺。這兩個問題的 GM 答案影響如何引導還原環節。建議對照 DM 手冊確認。

張猛（NPC）— 頁4–6（GM 指引）

張猛（NPC） 頁4–6 🟡 中（GM 用）
缺失：GM 紅色文字指引（NPC 有價值資訊）、人物關係密碼牌、玩家互動分配說明。
原因：頁面印刷品質差（字體過小、過淡），裝訂陰影疊加，幾乎全部無法讀取。建議以高解析度補拍，並確認密碼牌完整內容。

🟢 優先級3（基本完整，可選校對）

金四刃 頁2（開場段） 🟢 低
OCR 重建品質中等，部分詞語疑似誤讀，大意可理解。建議翻閱原本校對一遍。
整體最完整角色之一，補拍非必要。
嚴逸 頁5下、頁7下 🟢 低
環節說明下半遮蔽。嚴逸整體為全場最完整劇本，此缺失不影響遊戲執行。
整體最完整角色，補拍非緊迫。

📋 補拍操作說明

補拍後，用更新的圖片重新執行 OCR 提取，然後把新段落合併進對應的 md 整理稿中，最後重新生成 HTML 頁面。 詳細 OCR 提示詞模板請見 OCR 提示詞指引。

拍攝建議

  • 使用 PDF 掃描或手機拍攝，解析度 ≥ 300 DPI，建議 600 DPI。
  • 拍攝時盡量壓平書本內側，減少裝訂陰影。
  • 王思涵頁面建議以 2× 縮放 PNG 拍攝（印刷字體過小）。
  • 拍攝後命名格式：角色拼音-pXXX-retake.png，例如：jia-san-niang-p007-retake.png。
  • 存放位置：local-private/kou-xia/retake-photos/（此路徑已被 .gitignore 排除）。
