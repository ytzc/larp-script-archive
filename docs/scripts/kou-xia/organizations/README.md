# 組織詳細資料

> ⚠️ **含 Spoiler — 建議只有 GM 或已上桌的玩家才閱讀**
>
> 本資料夾包含各組織的完整內部資訊、隱藏關係與秘密動機。
> 公開版本的組織資訊請見 [`public/organizations.md`](../public/organizations.md)。

## 組織檔案列表

| 檔案 | 組織名稱 | 說明 |
|------|----------|------|
| [organization-template.md](organization-template.md) | 模板 | 新增組織時使用此模板 |

## 使用說明

1. 複製 `organization-template.md`，以組織縮寫命名（如 `org-jianghu-alliance.md`）。
2. 填入組織的完整資料，包含隱藏資訊。
3. 在此 README 更新列表。

## ⚠️ GitHub Pages 警告

若此 repo 部署至 GitHub Pages，`organizations/` 資料夾的完整內容將公開。
含 spoiler 的組織資料建議放在 private repo 或加入 `.gitignore`。
