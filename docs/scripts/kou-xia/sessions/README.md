## 使用說明

1. 複製 `session-template.md`，以日期命名，例如 `2026-06-20.md`。
2. 遊戲前填入玩家資訊與角色分配。
3. 遊戲後填入紀錄與心得。
4. 在此 README 更新列表。

## 可公開程度

場次記錄通常可以在遊戲結束後分享（不含 spoiler 的部分），請依個人需求決定是否公開。
